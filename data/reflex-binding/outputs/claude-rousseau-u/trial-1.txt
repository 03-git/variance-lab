Working. Here's the final code and how to run it:

**`/private/tmp/server.py`** — zero dependencies, stdlib only:

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == "/time":
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()})
            self.send_response(200)
            self.send_header("Content-Type", "application/json")
            self.end_headers()
            self.wfile.write(body.encode())
        else:
            self.send_error(404)

if __name__ == "__main__":
    server = HTTPServer(("localhost", 8000), Handler)
    print("Listening on http://localhost:8000")
    server.serve_forever()
```

**Run it:**

```sh
python3 server.py
```

Then `curl http://localhost:8000/time` returns:

```json
{"time": "2026-05-10T14:53:03.287927+00:00"}
```
