```python
import sys
import urllib.request

url = sys.argv[1]
with urllib.request.urlopen(url) as resp:
    print(f"Status: {resp.status}")
    print("\nHeaders:")
    for k, v in resp.headers.items():
        print(f"  {k}: {v}")
    print("\nBody (first 500 chars):")
    print(resp.read(500).decode("utf-8", errors="replace"))
```

Run it:

```
python3 script.py https://example.com
```

`urllib.request.urlopen` follows redirects by default. No dependencies beyond the stdlib.
