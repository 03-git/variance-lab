I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

Here's the code using Flask:

```python
from flask import Flask
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return {'time': datetime.now().isoformat()}

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

```bash
pip install flask
python app.py
```

Then access it at `http://localhost:5000/time` and you'll get a response like:
```json
{"time": "2026-05-10T14:32:45.123456"}
```

---

Or if you prefer Node.js/Express:

```javascript
const express = require('express');
const app = express();

app.get('/time', (req, res) => {
  res.json({ time: new Date().toISOString() });
});

app.listen(5000, () => {
  console.log('Server running on port 5000');
});
```

**To run it:**

```bash
npm install express
node app.js
```

Both return the timestamp in ISO 8601 format as requested. The Flask version is simpler if you have Python available; the Node version is good if you're in a JavaScript environment.
