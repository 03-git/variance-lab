I'll show you how to build a simple HTTP endpoint in Python that returns the current time as JSON.

Here's the code using Flask (lightweight and straightforward):

```python
from flask import Flask
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return {
        'time': datetime.utcnow().isoformat() + 'Z'
    }

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

```bash
pip install flask
python app.py
```

Then visit `http://localhost:5000/time` in your browser or curl it:

```bash
curl http://localhost:5000/time
```

You'll get a response like:
```json
{"time":"2026-05-10T14:32:45.123456Z"}
```

---

If you prefer using the standard library without external dependencies, here's a `http.server` version:

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            response = {
                'time': datetime.utcnow().isoformat() + 'Z'
            }
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, format, *args):
        pass  # Suppress request logging

if __name__ == '__main__':
    server = HTTPServer(('localhost', 5000), TimeHandler)
    print('Server running at http://localhost:5000')
    server.serve_forever()
```

**To run it:**

```bash
python app.py
```

The standard library version needs no dependencies. Both will return the current UTC time in ISO 8601 format with a `Z` suffix.
