I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            # Get current time in ISO 8601 format
            current_time = datetime.utcnow().isoformat() + 'Z'
            response = {'time': current_time}
            
            # Send response
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()
    
    def log_message(self, format, *args):
        pass  # Suppress default logging

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print('Server running at http://localhost:8000/time')
    server.serve_forever()
```

**To run it:**

```bash
python3 server.py
```

Then test it with:
```bash
curl http://localhost:8000/time
```

You'll get a response like:
```json
{"time": "2026-05-10T14:32:45.123456Z"}
```

**Alternative using Flask (if you prefer):**

```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return jsonify({'time': datetime.utcnow().isoformat() + 'Z'})

if __name__ == '__main__':
    app.run(port=8000)
```

The standard library version requires no dependencies, while Flask is simpler if you already have it installed.
