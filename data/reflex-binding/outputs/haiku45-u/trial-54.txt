I'll show you a simple HTTP endpoint in Python using Flask that returns the current time as JSON.

```python
from flask import Flask
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return {"time": datetime.now().isoformat()}

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

```bash
pip install flask
python app.py
```

Then visit `http://localhost:5000/time` in your browser or curl:

```bash
curl http://localhost:5000/time
```

You'll get a response like:
```json
{"time": "2026-05-10T14:23:45.123456"}
```

The `.isoformat()` method gives you the ISO 8601 format automatically. If you want to remove microseconds for a cleaner timestamp, use `datetime.now().isoformat(timespec='seconds')` instead.
