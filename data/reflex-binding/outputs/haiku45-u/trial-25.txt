I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

**Using Python's built-in `http.server`:**

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            response = {"time": datetime.utcnow().isoformat() + "Z"}
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, format, *args):
        pass  # Suppress log output

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print("Server running at http://localhost:8000/time")
    server.serve_forever()
```

This will output something like:
```json
{"time": "2026-05-10T14:32:45.123456Z"}
```

The endpoint listens on `localhost:8000`, responds to `GET /time` with the current UTC time in ISO 8601 format, and returns 404 for any other path.
