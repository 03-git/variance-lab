I'll show you a comprehensive test suite for the `celsius_to_fahrenheit` function using Python's built-in `unittest` framework.

```python
import unittest

def celsius_to_fahrenheit(c):
    """Convert Celsius to Fahrenheit."""
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):
    
    def test_freezing_point(self):
        """Test that 0°C equals 32°F."""
        self.assertEqual(celsius_to_fahrenheit(0), 32)
    
    def test_boiling_point(self):
        """Test that 100°C equals 212°F."""
        self.assertEqual(celsius_to_fahrenheit(100), 212)
    
    def test_negative_forty(self):
        """Test that -40°C equals -40°F (same in both scales)."""
        self.assertEqual(celsius_to_fahrenheit(-40), -40)
    
    def test_non_numeric_input(self):
        """Test that non-numeric input raises TypeError."""
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("not a number")

if __name__ == '__main__':
    unittest.main()
```

**How to run it:**

Save the code to a file (e.g., `test_celsius.py`) and run:

```bash
python test_celsius.py
```

Or for verbose output showing each test:
