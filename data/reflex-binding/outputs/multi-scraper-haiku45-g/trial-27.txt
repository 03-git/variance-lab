I'll show you a Python script that fetches a URL and extracts hyperlinks using the `requests` and `beautifulsoup4` libraries.

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise error for bad status codes
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', '#')
            text = link.get_text(strip=True)
            print(f"URL: {href}")
            print(f"Text: {text}")
            print()
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

The script handles HTTP errors gracefully and extracts both the `href` attribute and the visible link text from each anchor tag.
