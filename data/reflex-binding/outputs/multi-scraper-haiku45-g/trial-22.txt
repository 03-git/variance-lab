I'll show you a Python script that fetches a URL and extracts hyperlinks using the `requests` and `BeautifulSoup` libraries.

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com"

try:
    response = requests.get(url, timeout=10)
    response.raise_for_status()
    
    soup = BeautifulSoup(response.content, "html.parser")
    
    links = soup.find_all("a")
    
    if not links:
        print("No links found on the page.")
    else:
        print(f"Found {len(links)} link(s):\n")
        for link in links:
            href = link.get("href")
            text = link.get_text(strip=True)
            print(f"Text: {text}")
            print(f"URL:  {href}\n")
            
except requests.exceptions.RequestException as e:
    print(f"Error fetching the URL: {e}")
```

**To run it:**

1. Install dependencies if you don't have them:
   ```bash
   pip install requests beautifulsoup4
   ```

2. Run the script:
   ```bash
   python script.py
   ```

The script:
- Fetches the page with a 10-second timeout
- Parses the HTML with BeautifulSoup
- Finds all `<a>` tags
- Prints each link's text and href, with error handling for network issues

You can modify the `url` variable to scrape any website.
