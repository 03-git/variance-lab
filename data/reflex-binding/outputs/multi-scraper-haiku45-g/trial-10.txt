Here's a Python script to fetch a webpage and extract all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

# Fetch the webpage
response = requests.get('https://example.com')
response.raise_for_status()  # Raise an error for bad status codes

# Parse the HTML
soup = BeautifulSoup(response.content, 'html.parser')

# Find all anchor tags and extract href and text
for link in soup.find_all('a'):
    href = link.get('href')
    text = link.get_text(strip=True)
    
    if href:  # Only print if href exists
        print(f"Text: {text}")
        print(f"URL: {href}")
        print()
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- `requests.get()` fetches the webpage
- `BeautifulSoup` parses the HTML
- `.find_all('a')` finds all anchor tags
- `.get('href')` extracts the URL
- `.get_text(strip=True)` extracts the link text, removing whitespace

For `https://example.com` specifically, you'll see it has very few links since it's a placeholder domain, but this will work on any website.
