I'll show you a Python script that fetches a URL and extracts hyperlinks using the `requests` and `BeautifulSoup` libraries.

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise an error for bad status codes
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} link(s) on {url}\n")
        print(f"{'href':<50} | {'Link Text':<30}")
        print("-" * 82)
        
        # Extract and print href and link text
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            print(f"{href:<50} | {text:<30}")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script** (save it as `fetch_links.py`, then):
   ```bash
   python fetch_links.py
   ```

**What it does:**
- Fetches the URL using `requests`
- Parses the HTML with BeautifulSoup
- Extracts all `<a>` tags
- Prints the `href` attribute and the link text in a formatted table

You can modify the URL by changing the argument to `fetch_and_extract_links()` at the bottom, or make it accept command-line arguments if you want to make it more flexible.
