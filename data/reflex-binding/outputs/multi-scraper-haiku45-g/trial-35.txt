I'll show you a Python script that fetches a URL and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        # Fetch the page
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', 'No href')
            text = link.get_text(strip=True) or 'No text'
            print(f"Text: {text}")
            print(f"URL:  {href}\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the page: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**To run it:**

1. Install dependencies:
```bash
pip install requests beautifulsoup4
```

2. Run the script:
```bash
python script.py
```

The script fetches the page, parses the HTML, finds all `<a>` tags, and prints each link's text and href. It handles errors gracefully if the page can't be fetched.
