I'll show you a Python script that fetches a URL and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def extract_links(url):
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for i, link in enumerate(links, 1):
            href = link.get('href', '')
            text = link.get_text(strip=True)
            
            # Handle relative URLs
            if href:
                absolute_url = urljoin(url, href)
                print(f"{i}. Text: {text or '(no text)'}")
                print(f"   URL: {absolute_url}\n")
            else:
                print(f"{i}. Text: {text or '(no text)'}")
                print(f"   URL: (no href)\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. Install dependencies:
```bash
pip install requests beautifulsoup4
```

2. Run the script:
```bash
python script.py
```

**What it does:**
- Fetches the URL with a 10-second timeout
- Parses the HTML using BeautifulSoup
- Finds all `<a>` tags and extracts `href` attributes and link text
- Converts relative URLs to absolute URLs
- Prints each link with its number, text, and full URL

For `example.com`, you'll see the two links that are on that placeholder page.
