Here's a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise exception for bad status codes
        
        soup = BeautifulSoup(response.content, 'html.parser')
        
        links = soup.find_all('a')
        
        if not links:
            print("No links found.")
            return
        
        print(f"Found {len(links)} link(s):\n")
        for link in links:
            href = link.get('href', '#')
            text = link.get_text(strip=True)
            print(f"Text: {text}")
            print(f"URL:  {href}")
            print()
    
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the webpage using `requests`
- Parses the HTML with `BeautifulSoup`
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints each link with its URL and text
- Includes basic error handling for network issues

You can modify the URL by changing the argument in the `fetch_and_extract_links()` call at the bottom.
