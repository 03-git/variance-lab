Here's a Python script that fetches a URL and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        soup = BeautifulSoup(response.content, 'html.parser')
        
        links = soup.find_all('a')
        
        if not links:
            print("No links found.")
            return
        
        print(f"Found {len(links)} link(s):\n")
        for link in links:
            href = link.get('href')
            text = link.get_text(strip=True)
            print(f"  Text: {text}")
            print(f"  URL:  {href}")
            print()
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**To run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from the URL using `requests`
- Parses the HTML with BeautifulSoup
- Finds all `<a>` tags
- Prints the link text and href for each one
- Handles network errors gracefully

You can modify the URL by changing the argument to `extract_links()` at the bottom, or make it accept command-line arguments if you want more flexibility.
