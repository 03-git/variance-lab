I'll show you a Python script that fetches a URL and extracts hyperlinks using `requests` and `BeautifulSoup`:

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com"

try:
    response = requests.get(url, timeout=10)
    response.raise_for_status()
except requests.exceptions.RequestException as e:
    print(f"Error fetching {url}: {e}")
    exit(1)

soup = BeautifulSoup(response.content, "html.parser")

links = soup.find_all("a")

if not links:
    print("No links found.")
else:
    print(f"Found {len(links)} link(s):\n")
    for link in links:
        href = link.get("href")
        text = link.get_text(strip=True)
        print(f"Text: {text}")
        print(f"href: {href}\n")
```

**How to run it:**

1. Install dependencies (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. Run the script:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from `https://example.com`
- Parses it with BeautifulSoup
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints each link with its text and URL
- Handles network errors gracefully

You can modify the `url` variable to point to any website you want to scrape.
