Here's a Python script that fetches a URL and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return
    
    soup = BeautifulSoup(response.content, 'html.parser')
    
    links = soup.find_all('a')
    
    if not links:
        print("No links found.")
        return
    
    print(f"Found {len(links)} link(s):\n")
    for link in links:
        href = link.get('href')
        text = link.get_text(strip=True)
        print(f"Text: {text}")
        print(f"URL:  {href}\n")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies:**
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**To use with a different URL**, modify the last line:
```python
extract_links("https://your-url.com")
```

**What it does:**
- Fetches the HTML from the URL with error handling
- Parses the HTML using BeautifulSoup
- Finds all `<a>` tags
- Prints both the link text and href for each one

For `https://example.com` specifically, it will output the few informational links on that page.
