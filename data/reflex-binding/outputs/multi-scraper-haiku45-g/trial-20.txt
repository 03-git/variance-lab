Here's a Python script to fetch and extract hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

# Fetch the webpage
response = requests.get('https://example.com')
response.raise_for_status()

# Parse HTML
soup = BeautifulSoup(response.content, 'html.parser')

# Find and print all hyperlinks
for link in soup.find_all('a'):
    href = link.get('href')
    text = link.get_text(strip=True)
    if href:  # only print if href exists
        print(f"href: {href}")
        print(f"text: {text}")
        print()
```

**To run it:**

1. **Install dependencies** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Save and run** (e.g., as `fetch_links.py`):
   ```bash
   python fetch_links.py
   ```

**What it does:**
- `requests.get()` fetches the webpage
- `BeautifulSoup` parses the HTML
- `.find_all('a')` finds all anchor tags
- `.get('href')` and `.get_text()` extract the URL and link text
- The `strip=True` removes extra whitespace from link text

For example.com, you'll see output like:
```
href: https://www.iana.org/domains/example
text: More information...
```
