**Python (stdlib only — no dependencies):**

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != '/time':
            self.send_response(404)
            self.end_headers()
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header('Content-Type', 'application/json')
        self.send_header('Content-Length', len(body))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *_):
        pass

HTTPServer(('', 8080), Handler).serve_forever()
```

**Run:**
```bash
python3 time_server.py
```

**Test:**
```bash
curl http://localhost:8080/time
# {"time": "2026-05-10T14:23:01.456789+00:00"}
```

If you want something even more minimal as a one-liner:

```bash
python3 -c "
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class H(BaseHTTPRequestHandler):
    def do_GET(self):
        b = json.dumps({'time': datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200); self.send_header('Content-Type','application/json'); self.send_header('Content-Length',len(b)); self.end_headers(); self.wfile.write(b)
    def log_message(self,*_): pass

HTTPServer(('',8080),H).serve_forever()
"
```

Uses only stdlib, no pip required. Returns UTC time with timezone offset in ISO 8601.
