**Python (stdlib only):**

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(body)
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, *args):
        pass

HTTPServer(('', 8080), Handler).serve_forever()
```

Run it:

```bash
python3 time_server.py
```

Test it:

```bash
curl http://localhost:8080/time
# {"time": "2026-05-10T14:23:01.123456+00:00"}
```

---

If you want something shorter with no class boilerplate, here's the same thing as a one-liner using `http.server` via subshell isn't clean enough — stick with the above. Alternatively, if you have `flask` or `fastapi` installed:

**Flask:**

```python
from flask import Flask, jsonify
from datetime import datetime, timezone

app = Flask(__name__)

@app.get('/time')
def get_time():
    return jsonify({"time": datetime.now(timezone.utc).isoformat()})

app.run(port=8080)
```

```bash
python3 -m flask --app time_server run --port 8080
```

The stdlib version has zero dependencies and works as-is on this machine.
