**Python stdlib (no dependencies):**

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_response(404)
            self.end_headers()
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", len(body))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):
        pass  # suppress access logs

HTTPServer(("", 8080), Handler).serve_forever()
```

Run with:
```
python3 time_server.py
curl http://localhost:8080/time
```

**Flask (if you prefer):**

```python
from flask import Flask, jsonify
from datetime import datetime, timezone

app = Flask(__name__)

@app.get("/time")
def get_time():
    return jsonify({"time": datetime.now(timezone.utc).isoformat()})

if __name__ == "__main__":
    app.run(port=8080)
```

Run with:
```
pip install flask
python3 time_server.py
```

Both return something like `{"time": "2026-05-10T14:23:01.456789+00:00"}`. The stdlib version has zero deps; use Flask if you're already in a Flask project.
