**Python (stdlib only, no dependencies):**

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(body)
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, *args):
        pass  # suppress access logs

HTTPServer(('', 8080), Handler).serve_forever()
```

**Run:**

```bash
python3 time_server.py
```

**Test:**

```bash
curl http://localhost:8080/time
# {"time": "2026-05-10T14:23:01.123456+00:00"}
```

If you want something shorter and don't mind a dependency:

```bash
# Flask one-liner
python3 -c "
from flask import Flask, jsonify
from datetime import datetime, timezone
app = Flask(__name__)

@app.get('/time')
def t(): return jsonify(time=datetime.now(timezone.utc).isoformat())

app.run(port=8080)
"
```

The stdlib version has zero dependencies and works on any Python 3.x install.
