**Python (no dependencies):**

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(body)
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, *_):
        pass

HTTPServer(('', 8080), Handler).serve_forever()
```

**Run:**
```sh
python3 server.py
```

**Test:**
```sh
curl http://localhost:8080/time
# {"time": "2026-05-10T14:32:01.123456+00:00"}
```

If you want Flask instead (cleaner, needs `pip install flask`):

```python
from flask import Flask, jsonify
from datetime import datetime, timezone

app = Flask(__name__)

@app.get('/time')
def get_time():
    return jsonify({"time": datetime.now(timezone.utc).isoformat()})

app.run(port=8080)
```

The stdlib version has zero dependencies and runs anywhere Python 3 is present.
