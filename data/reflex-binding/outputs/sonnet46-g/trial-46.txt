**Python (stdlib only):**

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(body)
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, *args):
        pass  # silence access logs

HTTPServer(('', 8000), Handler).serve_forever()
```

**Run:**
```
python3 server.py
curl http://localhost:8000/time
```

**Response:**
```json
{"time": "2026-05-10T14:23:01.456789+00:00"}
```

If you want a one-liner instead:

```bash
python3 -c "
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class H(BaseHTTPRequestHandler):
    def do_GET(self):
        b = json.dumps({'time': datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200); self.send_header('Content-Type','application/json'); self.end_headers(); self.wfile.write(b)
    def log_message(self,*a): pass
HTTPServer(('',8000),H).serve_forever()
"
```

No dependencies beyond stdlib. Change `8000` to whatever port you need.
