I'll show you a Python CLI tool using `argparse` that meets your requirements:

```python
import argparse
import json
import sys

def main():
    parser = argparse.ArgumentParser(
        description="CLI tool to parse and display configuration as JSON"
    )
    parser.add_argument(
        "--host",
        required=True,
        help="Host address"
    )
    parser.add_argument(
        "--port",
        type=int,
        required=True,
        help="Port number (1-65535)"
    )
    parser.add_argument(
        "--verbose",
        action="store_true",
        help="Enable verbose output"
    )
    
    args = parser.parse_args()
    
    # Validate port range
    if not (1 <= args.port <= 65535):
        parser.error(f"Port must be between 1 and 65535, got {args.port}")
    
    # Build configuration dictionary
    config = {
        "host": args.host,
        "port": args.port,
        "verbose": args.verbose
    }
    
    # Output as JSON
    print(json.dumps(config, indent=2))

if __name__ == "__main__":
    main()
```

**How to run it:**

```bash
# Basic usage
python cli.py --host localhost --port 8080

# With verbose flag
python cli.py --host 0.0.0.0 --port 3000 --verbose

# Invalid port (will error)
python cli.py --host localhost --port 99999

# Help
python cli.py --help
```

**Example outputs:**

```bash
$ python cli.py --host localhost --port 8080
{
  "host": "localhost",
  "port": 8080,
  "verbose": false
}

$ python cli.py --host 0.0.0.0 --port 3000 --verbose
{
  "host": "0.0.0.0",
  "port": 3000,
  "verbose": true
}
```

The tool uses `argparse` to handle flag parsing, validates the port range with a conditional check after parsing, and outputs the configuration as formatted JSON.
