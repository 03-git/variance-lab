I'll show you a Python CLI tool that validates the flags and outputs JSON:

```python
import argparse
import json
import sys


def main():
    parser = argparse.ArgumentParser(
        description="Configuration validator",
        prog="config-tool"
    )
    parser.add_argument(
        "--host",
        type=str,
        default="localhost",
        help="Host to connect to (default: localhost)"
    )
    parser.add_argument(
        "--port",
        type=int,
        default=8080,
        help="Port number (default: 8080)"
    )
    parser.add_argument(
        "--verbose",
        action="store_true",
        help="Enable verbose output"
    )

    args = parser.parse_args()

    # Validate port range
    if not (1 <= args.port <= 65535):
        parser.error(f"port must be between 1 and 65535, got {args.port}")

    config = {
        "host": args.host,
        "port": args.port,
        "verbose": args.verbose
    }

    print(json.dumps(config, indent=2))


if __name__ == "__main__":
    main()
```

**How to run it:**

```bash
# Save as config_tool.py
python config_tool.py --host example.com --port 3000 --verbose

# Output:
# {
#   "host": "example.com",
#   "port": 3000,
#   "verbose": true
# }

# With defaults:
python config_tool.py

# Output:
# {
#   "host": "localhost",
#   "port": 8080,
#   "verbose": false
# }

# Invalid port (automatically caught):
python config_tool.py --port 99999
# Error: port must be between 1 and 65535, got 99999
```

The `argparse` module automatically:
- Converts `--port` to an integer (fails with a clear error if non-numeric)
- Handles `--verbose` as a boolean flag
- Provides `--help` for free
