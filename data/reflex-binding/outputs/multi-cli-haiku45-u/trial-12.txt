Here's a Python CLI tool that meets your requirements:

```python
import argparse
import json
import sys


def main():
    parser = argparse.ArgumentParser(
        description="CLI tool to parse and validate host/port configuration"
    )
    parser.add_argument("--host", default="localhost", help="Host address (default: localhost)")
    parser.add_argument("--port", type=int, help="Port number (1-65535)")
    parser.add_argument("--verbose", action="store_true", help="Enable verbose output")
    
    args = parser.parse_args()
    
    # Validate port
    if args.port is not None and (args.port < 1 or args.port > 65535):
        print(f"Error: Port must be between 1 and 65535, got {args.port}", file=sys.stderr)
        sys.exit(1)
    
    config = {
        "host": args.host,
        "port": args.port,
        "verbose": args.verbose
    }
    
    print(json.dumps(config, indent=2))


if __name__ == "__main__":
    main()
```

**How to run it:**

```bash
# Basic usage with defaults
python cli.py
# Output: {"host": "localhost", "port": null, "verbose": false}

# With all flags
python cli.py --host 0.0.0.0 --port 8080 --verbose
# Output: {"host": "0.0.0.0", "port": 8080, "verbose": true}

# Invalid port (will error)
python cli.py --port 99999
# Error: Port must be between 1 and 65535, got 99999

# Only host
python cli.py --host example.com
# Output: {"host": "example.com", "port": null, "verbose": false}
```

The tool uses `argparse` for parsing, validates the port range with a simple check, and outputs JSON with proper formatting.
