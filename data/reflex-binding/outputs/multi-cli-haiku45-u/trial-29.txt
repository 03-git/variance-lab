I'll show you a Python CLI tool that validates those flags and outputs JSON:

```python
import argparse
import json
import sys

def main():
    parser = argparse.ArgumentParser(
        description="CLI tool for host and port configuration"
    )
    parser.add_argument(
        "--host",
        type=str,
        default="localhost",
        help="Host to connect to (default: localhost)"
    )
    parser.add_argument(
        "--port",
        type=int,
        default=8000,
        help="Port number (default: 8000)"
    )
    parser.add_argument(
        "--verbose",
        action="store_true",
        help="Enable verbose output"
    )
    
    args = parser.parse_args()
    
    # Validate port range
    if not (1 <= args.port <= 65535):
        print(
            json.dumps({"error": f"Port must be between 1 and 65535, got {args.port}"}),
            file=sys.stderr
        )
        sys.exit(1)
    
    # Output configuration as JSON
    config = {
        "host": args.host,
        "port": args.port,
        "verbose": args.verbose
    }
    print(json.dumps(config, indent=2))

if __name__ == "__main__":
    main()
```

**How to run it:**

Save it as `cli_tool.py` and try these commands:

```bash
# Basic usage with defaults
python cli_tool.py

# With custom host and port
python cli_tool.py --host example.com --port 3000

# With verbose flag
python cli_tool.py --host 0.0.0.0 --port 5000 --verbose

# Invalid port (shows error)
python cli_tool.py --port 99999

# Help
python cli_tool.py --help
```

The tool uses `argparse` for clean flag parsing, validates the port is an integer in range [1, 65535], and outputs valid JSON to stdout (or an error to stderr if validation fails).
