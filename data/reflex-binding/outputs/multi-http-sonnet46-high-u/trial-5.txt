Here's a minimal HTTP server in Python using only the standard library:

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == "/time":
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
            self.send_response(200)
            self.send_header("Content-Type", "application/json")
            self.end_headers()
            self.wfile.write(body)
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, format, *args):
        pass  # suppress request logs

if __name__ == "__main__":
    server = HTTPServer(("", 8080), TimeHandler)
    print("Listening on http://localhost:8080")
    server.serve_forever()
```

**To run it:**

```bash
python server.py
```

**To test it:**

```bash
curl http://localhost:8080/time
```

**Example response:**

```json
{"time": "2026-05-10T14:23:01.456789+00:00"}
```

No dependencies beyond the Python standard library. The timestamp is UTC with timezone offset (`+00:00`) per ISO 8601.
