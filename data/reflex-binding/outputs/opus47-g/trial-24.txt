```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_response(404); self.end_headers(); return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

HTTPServer(("127.0.0.1", 8080), Handler).serve_forever()
```

Run: `python3 timeserver.py`, then `curl http://127.0.0.1:8080/time`.

Stdlib only — no pip, no framework. Kernel primitive is the listening socket; `http.server` is the thinnest wrapper over it that handles the HTTP parse.
