Kernel primitive: `http.server` (Python standard library)

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
import json
from datetime import datetime

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            
            response = {
                "time": datetime.now().isoformat()
            }
            self.wfile.write(json.dumps(response).encode('utf-8'))
        else:
            self.send_error(404, "Not Found")

if __name__ == '__main__':
    server_address = ('', 8000)
    httpd = HTTPServer(server_address, TimeHandler)
    print("Server running on http://localhost:8000")
    httpd.serve_forever()
```

### How to run

1. Save the code as `server.py`.
2. Run the server:
   ```bash
   python3 server.py
   ```
3. Test the endpoint in a new terminal:
   ```bash
   curl http://localhost:8000/time
   ```
