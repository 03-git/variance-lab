```python
import sqlite3
from datetime import datetime

conn = sqlite3.connect("notes.db")
cur = conn.cursor()

cur.execute("""
    CREATE TABLE IF NOT EXISTS notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
""")

now = datetime.now().isoformat(timespec="seconds")
samples = [
    ("Groceries", "Milk, eggs, bread", now),
    ("Ideas", "Write a SQLite demo script", now),
    ("Reminder", "Call the dentist on Monday", now),
]
cur.executemany(
    "INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)",
    samples,
)
conn.commit()

for row in cur.execute("SELECT id, title, body, created_at FROM notes"):
    print(row)

conn.close()
```

Run it:

```bash
python3 notes.py
```

`sqlite3` ships with the Python standard library, so no `pip install` is needed. The script creates `notes.db` in the current directory; delete it between runs if you don't want the sample rows to accumulate.
