Here's a Python script using `requests` and `BeautifulSoup`:

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")

for tag in soup.find_all("a"):
    href = tag.get("href", "")
    text = tag.get_text(strip=True)
    print(f"{text!r:30} -> {href}")
```

**How to run it:**

1. Install dependencies (one-time):
   ```bash
   pip install requests beautifulsoup4
   ```

2. Save the script (e.g. `links.py`) and run:
   ```bash
   python links.py
   ```

**What it does:**
- `requests.get` fetches the page HTML
- `BeautifulSoup` parses it
- `find_all("a")` collects every anchor tag
- For each tag, it prints the link text and `href` attribute side by side

For `https://example.com` specifically, you'll see one result — the "More information..." link pointing to `https://www.iana.org/domains/reserved`.
