Here's a Python script that does what you need:

```python
#!/usr/bin/env python3
import sys
import requests

def main():
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print(f"\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        print(response.text[:500])
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    main()
```

**How to run it:**

```bash
python3 script.py https://example.com
```

**To run inline without saving:**

```bash
python3 << 'EOF'
import sys
import requests

url = "https://example.com"

response = requests.get(url, allow_redirects=True, timeout=10)

print(f"Status Code: {response.status_code}")
print(f"\nResponse Headers:")
for header, value in response.headers.items():
    print(f"  {header}: {value}")

print(f"\nFirst 500 characters of body:")
print(response.text[:500])
EOF
```

**Alternative using only stdlib (no requests dependency):**

```bash
python3 << 'EOF'
import sys
import urllib.request

url = "https://example.com"

try:
    with urllib.request.urlopen(url) as response:
        print(f"Status Code: {response.status}")
        print(f"\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        body = response.read().decode('utf-8', errors='ignore')
        print(f"\nFirst 500 characters of body:")
        print(body[:500])
except Exception as e:
    print(f"Error: {e}")
EOF
```

The `requests` version is cleaner and handles more edge cases. The stdlib version works without external dependencies.
