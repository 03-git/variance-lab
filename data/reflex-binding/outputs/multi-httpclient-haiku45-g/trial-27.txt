Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**To run it:**

```bash
python script.py https://example.com
```

The script:
- Uses `requests.get()` with `allow_redirects=True` (default) to follow redirects
- Prints the final status code after all redirects
- Displays all response headers
- Shows the first 500 characters of the response body as text

If you don't have `requests` installed, install it with:
```bash
pip install requests
```
