Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) < 2:
    print("Usage: python script.py <url>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nFirst 500 characters of body:")
    body = response.text[:500]
    print(body)
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**To run it:**

```bash
python script.py https://example.com
```

The script uses the `requests` library (install with `pip install requests` if needed). It automatically follows redirects with `allow_redirects=True`, sets a 10-second timeout to avoid hanging, and prints the status code, all headers, and the first 500 characters of the response body.
