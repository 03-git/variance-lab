Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nHeaders:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nBody (first 500 characters):")
        print(response.text[:500])
        
    except requests.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
# Install requests if you don't have it
pip install requests

# Run the script with a URL
python script.py https://example.com

# Example with a URL that redirects
python script.py https://google.com
```

The script uses `requests.get()` with `allow_redirects=True` (the default) to automatically follow redirects. It prints the final status code, all response headers, and the first 500 characters of the response body.
