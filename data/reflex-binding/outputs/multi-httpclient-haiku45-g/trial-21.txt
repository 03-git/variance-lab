Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print(f"\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        if len(response.text) > 500:
            print("... (truncated)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    fetch_url(sys.argv[1])
```

**How to run it:**

```bash
python script.py "https://example.com"
```

Or with a redirect-heavy URL:

```bash
python script.py "https://bit.ly/some-short-url"
```

The script uses the `requests` library (which handles redirects automatically with `allow_redirects=True`), prints the final status code after all redirects, shows all response headers, and displays the first 500 characters of the response body.
