Here's a Python script that does what you need:

```python
#!/usr/bin/env python3
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <url>", file=sys.stderr)
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}", file=sys.stderr)
    sys.exit(1)
```

**To run it:**

```bash
python3 script.py https://example.com
```

The script:
- Takes the URL from the first command-line argument
- Uses `requests.get()` with `allow_redirects=True` to follow redirects automatically
- Prints the final status code, all response headers, and the first 500 characters of the response body
- Includes error handling for network issues
- Has a 10-second timeout to avoid hanging on unresponsive servers

If you don't have `requests` installed, install it with:
```bash
pip install requests
```
