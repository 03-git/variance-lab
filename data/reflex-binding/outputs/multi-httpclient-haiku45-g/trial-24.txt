Here's a Python script that does what you need:

```python
#!/usr/bin/env python3
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        # Print status code
        print(f"Status Code: {response.status_code}")
        
        # Print response headers
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        # Print first 500 characters of body
        print("\nResponse Body (first 500 characters):")
        body = response.text[:500]
        print(body)
        if len(response.text) > 500:
            print("...")
    
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    fetch_url(sys.argv[1])
```

**How to run it:**

```bash
python script.py https://example.com
```

Or if you save it as a file with executable permissions:

```bash
chmod +x script.py
./script.py https://example.com
```

The script uses the `requests` library (install with `pip install requests` if needed), automatically follows redirects, and handles errors gracefully.
