Here's a Python script that does exactly that:

```python
import sys
import requests

if len(sys.argv) < 2:
    print("Usage: python script.py <url>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}\n")
    
    print("Response Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**To run it:**

```bash
python script.py https://example.com
```

**What it does:**
- `requests.get()` with `allow_redirects=True` (default) automatically follows redirects
- Prints the final status code after all redirects
- Iterates through all response headers
- Prints the first 500 characters of the response body as text

**Requirements:**
Install requests if you don't have it:
```bash
pip install requests
```

The script will handle connection errors, timeouts, and invalid URLs gracefully with error messages.
