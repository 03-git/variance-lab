I'll write a Python script that makes a GET request and displays the information you need.

```python
#!/usr/bin/env python3
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        print(response.text[:500])
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    fetch_url(sys.argv[1])
```

**How to run it:**

```bash
python script.py https://example.com
```

The script uses the `requests` library (which handles redirects automatically with `allow_redirects=True`), sets a 10-second timeout, and prints the status code, all response headers, and the first 500 characters of the response body. If the URL is invalid or the request fails, it catches the exception and exits with an error message.
