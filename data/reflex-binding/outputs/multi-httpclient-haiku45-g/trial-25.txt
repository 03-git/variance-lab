I'll show you a Python script that does this:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print(f"\nHeaders:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nBody (first 500 characters):")
        body_preview = response.text[:500]
        print(body_preview)
        if len(response.text) > 500:
            print("...")
    
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    fetch_url(sys.argv[1])
```

**How to run it:**

```bash
python script.py https://example.com
```

Or with a URL that redirects:
```bash
python script.py https://bit.ly/some-link
```

The script uses the `requests` library (install with `pip install requests` if needed). It follows redirects automatically, prints the final status code, all response headers, and the first 500 characters of the response body.
