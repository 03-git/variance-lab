Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>", file=sys.stderr)
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nHeaders:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nBody (first 500 characters):")
    body_preview = response.text[:500]
    print(body_preview)
    if len(response.text) > 500:
        print("...")
        
except requests.exceptions.RequestException as e:
    print(f"Error: {e}", file=sys.stderr)
    sys.exit(1)
```

**How to run it:**

```bash
python script.py "https://example.com"
```

The script uses the `requests` library (which handles redirects automatically with `allow_redirects=True`), prints the status code, iterates through all response headers, and displays the first 500 characters of the response body with a truncation indicator if there's more content.
