Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) < 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
# Save as fetch.py, then:
python fetch.py https://example.com

# Or pass any URL:
python fetch.py https://github.com
```

**What it does:**
- `requests.get()` with `allow_redirects=True` (default) automatically follows redirects
- Prints the final status code after all redirects resolve
- Lists all response headers
- Shows the first 500 characters of the response body
- Catches and displays any network errors

**Requirements:** You'll need `requests` installed. If you don't have it:
```bash
pip install requests
```
