Here's a Python script that does what you need:

```python
#!/usr/bin/env python3
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print("\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        if len(response.text) > 500:
            print("... (truncated)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
# First, install requests if you don't have it
pip install requests

# Run the script
python script.py https://example.com
```

The script uses `requests.get()` with `allow_redirects=True` (the default), which automatically follows all redirects and returns the final response.
