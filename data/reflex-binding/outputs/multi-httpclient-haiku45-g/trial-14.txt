Here's a Python script that does exactly that:

```python
#!/usr/bin/env python3
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python3 script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}\n")
    
    print("Response Headers:")
    for header, value in response.headers.items():
        print(f"  {header}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python3 script.py "https://example.com"
```

**Example usage:**

```bash
python3 script.py "https://httpbin.org/status/302"
```

This will follow the redirect and show you the final response. If you need to use the standard library instead (no external dependencies), here's an alternative using `urllib`:

```python
#!/usr/bin/env python3
import sys
import urllib.request
import urllib.error

if len(sys.argv) != 2:
    print("Usage: python3 script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    with urllib.request.urlopen(url, timeout=10) as response:
        print(f"Status Code: {response.status}\n")
        
        print("Response Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        body = response.read().decode('utf-8', errors='ignore')
        print(f"\nFirst 500 characters of body:")
        print(body[:500])
        
except urllib.error.URLError as e:
    print(f"Error: {e}")
    sys.exit(1)
```

The `requests` version is cleaner and handles edge cases better. The `urllib` version requires no external packages.
