```python
import pytest
from temperature import celsius_to_fahrenheit


def test_freezing():
    assert celsius_to_fahrenheit(0) == 32


def test_boiling():
    assert celsius_to_fahrenheit(100) == 212


def test_negative_forty_crossover():
    assert celsius_to_fahrenheit(-40) == -40


def test_non_numeric_raises():
    with pytest.raises(TypeError):
        celsius_to_fahrenheit("hot")
```

Run with:

```
pytest test_temperature.py -v
```

Assumes `celsius_to_fahrenheit` lives in `temperature.py` alongside the test file. The non-numeric case relies on Python's built-in `TypeError` from `*` on a `str` — no validation needed in the function itself.
