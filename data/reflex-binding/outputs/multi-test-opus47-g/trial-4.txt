```python
import pytest
from temperature import celsius_to_fahrenheit


def test_freezing():
    assert celsius_to_fahrenheit(0) == 32

def test_boiling():
    assert celsius_to_fahrenheit(100) == 212

def test_minus_forty_crossover():
    assert celsius_to_fahrenheit(-40) == -40

def test_non_numeric_raises():
    with pytest.raises(TypeError):
        celsius_to_fahrenheit("hot")
```

Run with:

```
pip install pytest
pytest test_temperature.py -v
```

Assumes `celsius_to_fahrenheit` lives in `temperature.py`. The `-40` test is the fixed point where the two scales coincide. The non-numeric test relies on Python's built-in `TypeError` from `*` on a `str`; no input validation needed in the function.
