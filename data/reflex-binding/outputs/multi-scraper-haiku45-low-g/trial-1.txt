I'll show you a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def extract_links(url):
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Links found on {url}:\n")
        for link in links:
            href = link.get('href')
            text = link.get_text(strip=True)
            
            # Convert relative URLs to absolute
            if href:
                href = urljoin(url, href)
            
            print(f"URL: {href}")
            print(f"Text: {text}")
            print("-" * 50)
    
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. Install required packages:
```bash
pip install requests beautifulsoup4
```

2. Save the script to a file (e.g., `extract_links.py`) and run:
```bash
python extract_links.py
```

Or run it directly:
```bash
python -c "$(cat <<'EOF'
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

response = requests.get("https://example.com", timeout=10)
soup = BeautifulSoup(response.content, 'html.parser')

for link in soup.find_all('a'):
    href = link.get('href')
    text = link.get_text(strip=True)
    if href:
        href = urljoin("https://example.com", href)
    print(f"URL: {href}\nText: {text}\n")
EOF
)"
```

The script handles relative URLs by converting them to absolute URLs using `urljoin()`, so you get complete links regardless of how they're written in the HTML.
