```python
import sys
import urllib.request

url = sys.argv[1]
with urllib.request.urlopen(url) as resp:
    body = resp.read(500).decode("utf-8", errors="replace")
    print(f"Status: {resp.status}")
    print("Headers:")
    for k, v in resp.headers.items():
        print(f"  {k}: {v}")
    print("Body (first 500 chars):")
    print(body)
```

Run: `python3 script.py https://example.com`

`urllib.request.urlopen` follows HTTP redirects by default via `HTTPRedirectHandler`. No third-party dependencies.
