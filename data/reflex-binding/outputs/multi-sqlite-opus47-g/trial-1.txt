```python
import sqlite3
from datetime import datetime

conn = sqlite3.connect("notes.db")
cur = conn.cursor()

cur.execute("""
    CREATE TABLE IF NOT EXISTS notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
""")

now = datetime.now().isoformat(timespec="seconds")
samples = [
    ("kernel first", "ask what the kernel calls it before reaching for a wrapper", now),
    ("via negativa", "subtract before adding", now),
    ("verify before acting", "read state, do not infer it", now),
]
cur.executemany("INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)", samples)
conn.commit()

for row in cur.execute("SELECT id, title, body, created_at FROM notes"):
    print(row)

conn.close()
```

Run:
```
python3 notes.py
```

Re-running appends another 3 rows since the table persists. `rm notes.db` between runs to start fresh.
