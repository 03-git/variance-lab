```python
import sqlite3
from datetime import datetime, timezone

conn = sqlite3.connect("notes.db")
cur = conn.cursor()

cur.execute("""
    CREATE TABLE IF NOT EXISTS notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
""")

now = datetime.now(timezone.utc).isoformat()
samples = [
    ("first", "kernel before wrapper", now),
    ("second", "verify before acting", now),
    ("third", "subtract, don't add", now),
]
cur.executemany(
    "INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)",
    samples,
)
conn.commit()

for row in cur.execute("SELECT id, title, body, created_at FROM notes"):
    print(row)

conn.close()
```

Run:
```
python3 notes.py
```

Creates `notes.db` in the cwd. Re-runs append 3 more rows each time; `rm notes.db` to reset.
