

```python
import urllib.request
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self._href = None
        self._text = []

    def handle_starttag(self, tag, attrs):
        if tag == "a":
            self._href = dict(attrs).get("href")
            self._text = []

    def handle_data(self, data):
        if self._href is not None:
            self._text.append(data)

    def handle_endtag(self, tag):
        if tag == "a" and self._href is not None:
            print(f"{self._href}  ->  {''.join(self._text).strip()}")
            self._href = None

with urllib.request.urlopen("https://example.com") as r:
    LinkExtractor().feed(r.read().decode())
```

Run it:

```
python3 script.py
```

Uses only stdlib (`urllib.request` for fetching, `html.parser` for parsing). No pip installs needed.
