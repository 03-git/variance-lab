```python
import sqlite3
from datetime import datetime

conn = sqlite3.connect("notes.db")
cur = conn.cursor()

cur.execute("""
CREATE TABLE IF NOT EXISTS notes (
    id INTEGER PRIMARY KEY,
    title TEXT,
    body TEXT,
    created_at TEXT
)
""")

now = datetime.utcnow().isoformat(timespec="seconds")
samples = [
    ("kernel first", "ask what the kernel calls it before reaching for a wrapper", now),
    ("verify before act", "live read beats remembered state", now),
    ("subtract", "less wrapper, more primitive", now),
]
cur.executemany("INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)", samples)
conn.commit()

for row in cur.execute("SELECT id, title, body, created_at FROM notes"):
    print(row)

conn.close()
```

Run: `python3 script.py`

Re-running appends 3 more rows each time (no dedupe). Delete `notes.db` to reset.
