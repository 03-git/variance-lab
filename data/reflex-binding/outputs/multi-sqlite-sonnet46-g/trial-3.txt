```python
import sqlite3
from datetime import datetime

con = sqlite3.connect(":memory:")
cur = con.cursor()

cur.execute("""
    CREATE TABLE notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
""")

notes = [
    ("Shopping list", "Eggs, milk, bread", datetime.now().isoformat()),
    ("Meeting notes", "Discuss Q3 roadmap with team", datetime.now().isoformat()),
    ("Book recommendation", "Read 'The Dispossessed' by Le Guin", datetime.now().isoformat()),
]

cur.executemany("INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)", notes)
con.commit()

for row in cur.execute("SELECT * FROM notes"):
    id_, title, body, created_at = row
    print(f"[{id_}] {title}")
    print(f"     {body}")
    print(f"     {created_at}")
    print()

con.close()
```

**To run:**

```bash
python3 notes.py
```

This uses an in-memory database (`:memory:`), so nothing is written to disk. To persist, replace `":memory:"` with a file path like `"notes.db"`.
