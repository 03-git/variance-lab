```python
import unittest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):

    def test_freezing_point(self):
        self.assertEqual(celsius_to_fahrenheit(0), 32)

    def test_boiling_point(self):
        self.assertEqual(celsius_to_fahrenheit(100), 212)

    def test_negative_forty(self):
        # -40 is the point where Celsius and Fahrenheit are equal
        self.assertEqual(celsius_to_fahrenheit(-40), -40)

    def test_non_numeric_input_raises(self):
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("hot")

if __name__ == "__main__":
    unittest.main()
```

**To run it:**

Save as `test_temp.py`, then:

```bash
python test_temp.py
```

Or with verbose output:

```bash
python -m unittest test_temp -v
```

**Expected output:**

```

test_non_numeric_input_raises (test_temp.TestCelsiusToFahrenheit) ... ok

----------------------------------------------------------------------

```

One note: the `TypeError` test works because Python raises `TypeError` when multiplying a string by a float (`"hot" * 9/5`). No extra validation code needed in the function itself.
