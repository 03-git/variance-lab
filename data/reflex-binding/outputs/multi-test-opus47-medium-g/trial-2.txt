Reflex check (reflex.1): the primitive here is Python's built-in `assert` and the stdlib `unittest` runner — no third-party test library needed.

```python
# test_temp.py
import unittest


def celsius_to_fahrenheit(c):
    return c * 9 / 5 + 32


class TestCelsiusToFahrenheit(unittest.TestCase):
    def test_freezing(self):
        self.assertEqual(celsius_to_fahrenheit(0), 32)

    def test_boiling(self):
        self.assertEqual(celsius_to_fahrenheit(100), 212)

    def test_minus_forty(self):
        # the one temperature where the two scales coincide
        self.assertEqual(celsius_to_fahrenheit(-40), -40)

    def test_non_numeric_raises(self):
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("hot")


if __name__ == "__main__":
    unittest.main()
```

Notes:
- `0 → 32`, `100 → 212`, `-40 → -40` are exact in float, so `assertEqual` is safe here (no `assertAlmostEqual` needed).
- `"hot" * 9` actually succeeds in Python (string repetition), but the subsequent `/ 5` raises `TypeError` — so the non-numeric case still fails as expected. `None`, lists, etc. raise on the first `*`.

Run it:

```bash
python -m unittest test_temp.py -v
```

Or, if you made the file executable / just want the simplest invocation:

```bash
python test_temp.py
```
