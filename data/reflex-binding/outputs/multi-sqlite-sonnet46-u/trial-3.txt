```python
import sqlite3
from datetime import datetime

conn = sqlite3.connect("notes.db")
cur = conn.cursor()

cur.execute("""
    CREATE TABLE IF NOT EXISTS notes (
        id         INTEGER PRIMARY KEY,
        title      TEXT,
        body       TEXT,
        created_at TEXT
    )
""")

notes = [
    ("Meeting Notes", "Discuss Q3 roadmap and team allocation.", datetime.now().isoformat()),
    ("Shopping List", "Milk, eggs, bread, coffee.", datetime.now().isoformat()),
    ("Book Recommendation", "Read 'The Pragmatic Programmer' next.", datetime.now().isoformat()),
]

cur.executemany(
    "INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)",
    notes,
)
conn.commit()

cur.execute("SELECT id, title, body, created_at FROM notes")
for row in cur.fetchall():
    id_, title, body, created_at = row
    print(f"[{id_}] {title} ({created_at})\n    {body}\n")

conn.close()
```

**How to run:**

```bash
python notes.py
```

**Expected output:**
```
[1] Meeting Notes (2026-05-10T...)
    Discuss Q3 roadmap and team allocation.

[2] Shopping List (2026-05-10T...)
    Milk, eggs, bread, coffee.

[3] Book Recommendation (2026-05-10T...)
    Read 'The Pragmatic Programmer' next.
```

A `notes.db` file will be created in the current directory. Re-running the script will insert duplicate rows since there's no deduplication — add `DROP TABLE IF EXISTS notes` before the `CREATE TABLE` if you want a clean run each time.
