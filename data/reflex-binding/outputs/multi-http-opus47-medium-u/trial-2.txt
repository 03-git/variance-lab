Here's a self-contained Python implementation using only the standard library — no dependencies, no files needed beyond the one you'd save it as (your call, I'm not writing it):

```python
# time_server.py
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404, "Not Found")
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # silence default request logging
        pass

if __name__ == "__main__":
    HTTPServer(("0.0.0.0", 8000), TimeHandler).serve_forever()
```

Run it:

```sh
python3 time_server.py
```

Test it:

```sh
curl http://localhost:8000/time
# {"time": "2026-05-10T17:42:03.512345+00:00"}
```

Notes:
- Uses UTC; `isoformat()` emits a valid ISO 8601 string with offset. If you'd rather have the trailing `Z` form, swap to `datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")`.
- Anything other than `GET /time` returns 404.
- Single-threaded; fine for local use. For concurrency use `ThreadingHTTPServer` instead of `HTTPServer` (same stdlib module).
