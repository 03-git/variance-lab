

```python
import urllib.request
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self._href = None
        self._text = []

    def handle_starttag(self, tag, attrs):
        if tag == "a":
            for name, value in attrs:
                if name == "href":
                    self._href = value
                    self._text = []

    def handle_data(self, data):
        if self._href is not None:
            self._text.append(data)

    def handle_endtag(self, tag):
        if tag == "a" and self._href is not None:
            self.links.append((self._href, "".join(self._text).strip()))
            self._href = None
            self._text = []

html = urllib.request.urlopen("https://example.com").read().decode()
parser = LinkExtractor()
parser.feed(html)

for href, text in parser.links:
    print(f"{href}  ->  {text}")
```

Run it:

```
python3 script.py
```

Zero dependencies — uses only `urllib.request` and `html.parser` from the standard library. The `LinkExtractor` subclass walks the HTML, captures each `<a href="...">` and its enclosed text, then prints them as `href -> text` pairs.
