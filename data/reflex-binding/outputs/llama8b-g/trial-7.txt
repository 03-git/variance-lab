**time_endpoint.py**
```python
from http.server import BaseHTTPRequestHandler, HTTPServer
import json
from datetime import datetime

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        self.send_response(200)
        self.send_header('Content-type', 'application/json')
        self.end_headers()
        current_time = datetime.utcnow().isoformat() + 'Z'
        self.wfile.write(json.dumps({'time': current_time}).encode())

def run_server():
    server_address = ('', 8000)
    httpd = HTTPServer(server_address, TimeHandler)
    print('Server running at http://localhost:8000/time')
    httpd.serve_forever()

if __name__ == '__main__':
    run_server()
```

**Running the code:**

1. Save the above code in a file named `time_endpoint.py`.
2. Open a terminal and navigate to the directory where you saved the file.
3. Install the `http.server` module if you haven't already: `pip install http.server`
4. Run the server using Python: `python time_endpoint.py`
5. Open a web browser and navigate to `http://localhost:8000/time`
6. You should see a JSON response with the current time in ISO 8601 format, e.g. `{"time": "2023-03-09T14:30:00Z"}`

Note: This code uses the built-in `http.server` module in Python, which is a simple HTTP server. It's not recommended for production use, but it's fine for development and testing purposes. For a more robust and secure HTTP server, consider using a WSGI server like Gunicorn or uWSGI.
