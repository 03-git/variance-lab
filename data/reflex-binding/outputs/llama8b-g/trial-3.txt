**time.py**
```python
from http.server import BaseHTTPRequestHandler, HTTPServer
import json
from datetime import datetime

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        self.send_response(200)
        self.send_header('Content-type', 'application/json')
        self.end_headers()
        timestamp = datetime.now().isoformat()
        self.wfile.write(json.dumps({'time': timestamp}).encode('utf-8'))

def run_server():
    server_address = ('', 8000)
    httpd = HTTPServer(server_address, TimeHandler)
    print('Server running at http://localhost:8000/time')
    httpd.serve_forever()

if __name__ == '__main__':
    run_server()
```
**Running the server:**

1. Save the code above in a file named `time.py`.
2. Open a terminal and navigate to the directory where you saved the file.
3. Install the `http.server` module if you haven't already: `pip install http.server` (note: this is a built-in module, so you might not need to install it).
4. Run the server: `python time.py`
5. Open a web browser and navigate to `http://localhost:8000/time`
6. You should see a JSON response with the current time in ISO 8601 format: `{"time": "2023-03-16T14:30:00.000000Z"}`

**Tips:**

* You can also use a tool like `curl` to test the endpoint from the command line: `curl http://localhost:8000/time`
* To stop the server, press `Ctrl+C` in the terminal where it's running.
* You can modify the server to run on a different port or address by changing the `server_address` variable in the `run_server` function.
