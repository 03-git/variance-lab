Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <url>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    body_preview = response.text[:500]
    print(f"\nFirst 500 characters of body:")
    print(body_preview)
    if len(response.text) > 500:
        print("...")
        
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**To run it:**

```bash
python script.py "https://example.com"
```

**What it does:**
- Takes a URL from the command line
- Makes a GET request with `allow_redirects=True` (follows redirects automatically)
- Prints the final status code
- Prints all response headers
- Prints the first 500 characters of the response body
- Handles network errors gracefully

**Requirements:**
Make sure you have `requests` installed:
```bash
pip install requests
```
