Here's a Python script that does what you need:

```python
#!/usr/bin/env python3
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>", file=sys.stderr)
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}", file=sys.stderr)
    sys.exit(1)
```

**To run it:**

```bash
python3 script.py "https://example.com"
```

The script uses the `requests` library (install with `pip install requests` if needed). It:
- Follows redirects automatically with `allow_redirects=True`
- Sets a 10-second timeout
- Prints the status code, all response headers, and the first 500 characters of the response body
- Handles request errors gracefully
