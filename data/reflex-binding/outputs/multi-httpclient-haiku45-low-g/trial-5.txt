Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print(f"\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    body_preview = response.text[:500]
    print(f"\nFirst 500 characters of body:")
    print(body_preview)
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**To run it:**

```bash
python script.py https://example.com
```

The script:
- Takes a URL as the first command-line argument
- Uses `requests.get()` with `allow_redirects=True` to follow redirects automatically
- Prints the final status code (after all redirects)
- Prints all response headers
- Prints the first 500 characters of the response body
- Includes a 10-second timeout and error handling for network issues

If you need to handle binary responses differently, you can use `response.content[:500]` instead of `response.text[:500]`.
