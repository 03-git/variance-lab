```python
import asyncio

async def handle_client(reader: asyncio.StreamReader, writer: asyncio.StreamWriter) -> None:
    peer = writer.get_extra_info("peername")
    print(f"Connected: {peer}")
    try:
        while True:
            data = await reader.readline()
            if not data:  # client closed the connection
                break
            writer.write(data)
            await writer.drain()
            print(f"Echoed to {peer}: {data!r}")
    except ConnectionResetError:
        pass
    finally:
        print(f"Disconnected: {peer}")
        writer.close()
        await writer.wait_closed()

async def main(host: str = "127.0.0.1", port: int = 8888) -> None:
    server = await asyncio.start_server(handle_client, host, port)
    addrs = ", ".join(str(s.getsockname()) for s in server.sockets)
    print(f"Serving on {addrs}")
    async with server:
        await server.serve_forever()

if __name__ == "__main__":
    try:
        asyncio.run(main())
    except KeyboardInterrupt:
        print("\nShutting down.")
```

## How it works

- `asyncio.start_server` accepts connections and spawns `handle_client` as a task per client, so many clients are served concurrently on a single thread.
- `reader.readline()` waits for a `\n`-terminated line. An empty result means EOF (client hung up), so we exit the loop.
- `writer.write()` + `await writer.drain()` sends the line back and applies backpressure if the socket buffer is full.

## Running it

1. Save as `echo_server.py` and start it:
   ```bash
   python echo_server.py
   ```

2. Connect with any TCP client. Using netcat:
   ```bash
   nc 127.0.0.1 8888
   ```
   Type a line, press Enter, and it comes right back. Open several terminals to see concurrent clients.

3. Or test with Python:
   ```python
   import asyncio

   async def test():
       reader, writer = await asyncio.open_connection("127.0.0.1", 8888)
       writer.write(b"hello world\n")
       await writer.drain()
       print(await reader.readline())  # b'hello world\n'
       writer.close()
       await writer.wait_closed()

   asyncio.run(test())
   ```

Stop the server with Ctrl+C.

Note: `readline()` accumulates until it sees a newline — if a client sends data without ever sending `\n`, that read won't complete. For untrusted input you'd want `readuntil` with a size limit or `read(n)` with your own buffering.
