Here's a complete async TCP echo server using Python's `asyncio`:

```python
import asyncio

async def handle_client(reader, writer):
    """Handle a single client connection"""
    addr = writer.get_extra_info('peername')
    print(f"Client connected: {addr}")
    
    try:
        while True:
            # Read a line from the client
            data = await reader.readuntil(b'\n')
            if not data:
                break
            
            message = data.decode().strip()
            print(f"[{addr}] Received: {message}")
            
            # Echo it back
            writer.write(data)
            await writer.drain()
            
    except asyncio.IncompleteReadError:
        pass
    except Exception as e:
        print(f"Error handling {addr}: {e}")
    finally:
        writer.close()
        await writer.wait_closed()
        print(f"Client disconnected: {addr}")

async def main():
    """Start the echo server"""
    server = await asyncio.start_server(
        handle_client, 
        '127.0.0.1', 
        8888
    )
    
    addr = server.sockets[0].getsockname()
    print(f"Server listening on {addr}")
    
    async with server:
        await server.serve_forever()

if __name__ == "__main__":
    asyncio.run(main())
```

**To run the server:**

```bash
python echo_server.py
```

**To test it, in another terminal:**

```bash
# Using telnet
telnet localhost 8888

# Or using netcat
nc localhost 8888

# Or using a Python client
python -c "
import socket
s = socket.socket()
s.connect(('localhost', 8888))
s.send(b'Hello, server!\n')
print(s.recv(1024).decode())
s.close()
"
```

**How it works:**

- `asyncio.start_server()` creates a server that accepts incoming connections
- `handle_client()` is an async coroutine that processes each client
- `reader.readuntil(b'\n')` waits for a newline-terminated message from the client
- `writer.write()` sends the data back, and `await writer.drain()` ensures it's flushed
- Multiple clients are handled concurrently—each gets its own `handle_client()` coroutine running in parallel

The server will print connection/disconnection messages and echo all client messages to stdout.
