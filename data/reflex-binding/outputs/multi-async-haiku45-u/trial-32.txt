I'll show you an async TCP echo server using Python's `asyncio` module that handles multiple concurrent clients efficiently.

```python
import asyncio

async def handle_client(reader, writer):
    """Handle a single client connection."""
    addr = writer.get_extra_info('peername')
    print(f"Client connected: {addr}")
    
    try:
        while True:
            # Read a line from the client
            data = await reader.readline()
            
            # If no data, client disconnected
            if not data:
                break
            
            # Echo the data back
            writer.write(data)
            await writer.drain()
            
            print(f"Echo from {addr}: {data.decode().strip()}")
    
    except asyncio.CancelledError:
        print(f"Client {addr} connection cancelled")
    except Exception as e:
        print(f"Error with client {addr}: {e}")
    finally:
        writer.close()
        await writer.wait_closed()
        print(f"Client disconnected: {addr}")

async def main():
    """Start the TCP echo server."""
    server = await asyncio.start_server(
        handle_client,
        '127.0.0.1',
        8888
    )
    
    addr = server.sockets[0].getsockname()
    print(f"Echo server listening on {addr}")
    
    async with server:
        await server.serve_forever()

if __name__ == "__main__":
    asyncio.run(main())
```

**How to run it:**

1. **Start the server** in one terminal:
```bash
python echo_server.py
```
Output:
```
Echo server listening on ('127.0.0.1', 8888)
```

2. **Connect clients** in other terminals using `nc` (netcat):
```bash
nc localhost 8888
```

3. **Send text** from any client:
```
hello
world
test message
```

Each line you type will be echoed back immediately.

**Key features:**
- **`asyncio.start_server()`** — Sets up the server to accept multiple concurrent connections
- **`handle_client()`** — Coroutine that runs for each client; uses `readline()` to read line-terminated input
- **`await writer.drain()`** — Ensures data is flushed before continuing
- **Exception handling** — Gracefully closes connections on error or client disconnect
- **Concurrent** — Handles many clients simultaneously without blocking

To test with multiple clients at once:
```bash
# Terminal 1: Start server
python echo_server.py

# Terminal 2, 3, 4, etc: Connect clients
nc localhost 8888
```

Type in any terminal and all clients receive their own echoed lines independently.
