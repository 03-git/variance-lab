I'll show you an async TCP echo server using Python's `asyncio` module:

```python
import asyncio

async def handle_client(reader, writer):
    """Handle a single client connection."""
    addr = writer.get_extra_info('peername')
    print(f"Client connected: {addr}")
    
    try:
        while True:
            # Read a line from the client
            data = await reader.readline()
            
            if not data:
                # Client disconnected
                break
            
            # Echo the data back
            writer.write(data)
            await writer.drain()
            
            print(f"Echo from {addr}: {data.decode().strip()}")
    except asyncio.CancelledError:
        pass
    finally:
        writer.close()
        await writer.wait_closed()
        print(f"Client disconnected: {addr}")

async def start_server():
    """Start the echo server."""
    server = await asyncio.start_server(
        handle_client,
        '127.0.0.1',
        8888
    )
    
    async with server:
        print("Echo server listening on 127.0.0.1:8888")
        await server.serve_forever()

if __name__ == "__main__":
    asyncio.run(start_server())
```

**How to run it:**

1. **Start the server in one terminal:**
   ```bash
   python echo_server.py
   ```

2. **Connect a client in another terminal:**
   ```bash
   nc localhost 8888
   ```
   or
   ```bash
   telnet localhost 8888
   ```

3. **Type messages** and they'll be echoed back. Type `Ctrl+C` to disconnect.

**What it does:**
- Uses `asyncio.start_server()` to create a TCP server on port 8888
- `handle_client()` runs concurrently for each connected client
- Reads lines with `reader.readline()` and echoes them back with `writer.write()`
- Properly handles client disconnections and cleanup with `wait_closed()`

You can connect multiple clients simultaneously and each will be handled independently in separate async tasks.
