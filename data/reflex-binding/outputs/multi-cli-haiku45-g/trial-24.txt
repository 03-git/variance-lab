Here's a Python CLI tool that meets your requirements:

```python
import argparse
import json
import sys


def validate_port(value):
    """Validate that port is an integer between 1-65535."""
    try:
        port = int(value)
        if not (1 <= port <= 65535):
            raise argparse.ArgumentTypeError(
                f"port must be between 1 and 65535, got {port}"
            )
        return port
    except ValueError:
        raise argparse.ArgumentTypeError(f"port must be an integer, got {value}")


def main():
    parser = argparse.ArgumentParser(
        description="Parse and display host/port configuration"
    )
    parser.add_argument("--host", default="localhost", help="Host to connect to")
    parser.add_argument(
        "--port",
        type=validate_port,
        default=8080,
        help="Port number (1-65535, default: 8080)",
    )
    parser.add_argument(
        "--verbose",
        action="store_true",
        help="Enable verbose output",
    )

    args = parser.parse_args()

    config = {
        "host": args.host,
        "port": args.port,
        "verbose": args.verbose,
    }

    print(json.dumps(config, indent=2))


if __name__ == "__main__":
    main()
```

**How to run it:**

```bash
# Basic usage (uses defaults)
python script.py

# With custom host and port
python script.py --host example.com --port 3000

# With verbose flag
python script.py --host 0.0.0.0 --port 5000 --verbose

# Invalid port (will error)
python script.py --port 99999

# Invalid port type (will error)
python script.py --port abc
```

**Example outputs:**

```bash
$ python script.py
{
  "host": "localhost",
  "port": 8080,
  "verbose": false
}

$ python script.py --host api.example.com --port 443 --verbose
{
  "host": "api.example.com",
  "port": 443,
  "verbose": true
}
```

The key features:
- `validate_port()` custom type function validates the port is a valid integer in range
- `argparse` handles parsing and generates helpful error messages
- JSON output for easy parsing downstream
- Sensible defaults: localhost, 8080, verbose off
