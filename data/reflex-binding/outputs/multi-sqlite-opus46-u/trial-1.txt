```python
import sqlite3
from datetime import datetime, timezone

# Connect (creates notes.db if it doesn't exist)
conn = sqlite3.connect("notes.db")
cur = conn.cursor()

# Create the notes table
cur.execute("""
    CREATE TABLE IF NOT EXISTS notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
""")

# Insert 3 sample notes
now = datetime.now(timezone.utc).isoformat()
sample_notes = [
    ("Shopping list", "Milk, eggs, bread, coffee", now),
    ("Meeting notes", "Discuss Q3 roadmap and hiring plan", now),
    ("Idea", "Build a CLI tool for managing notes", now),
]
cur.executemany(
    "INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)",
    sample_notes,
)
conn.commit()

# Query all notes
cur.execute("SELECT id, title, body, created_at FROM notes")
rows = cur.fetchall()

# Print them
for row in rows:
    note_id, title, body, created_at = row
    print(f"[{note_id}] {title} ({created_at})")
    print(f"    {body}")

conn.close()
```

How to run it:

1. Save the code to a file, e.g. `notes_demo.py`.
2. Run it with Python 3 (no dependencies needed — `sqlite3` is in the standard library):

```bash
python3 notes_demo.py
```

Expected output (timestamps will vary):


Note: each run appends 3 more notes to `notes.db`. Delete the file (`rm notes.db`) between runs for a clean slate, or change `CREATE TABLE` setup as needed.
