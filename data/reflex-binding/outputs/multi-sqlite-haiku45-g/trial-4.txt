Here's a Python script that creates a SQLite database with notes:

```python
import sqlite3
from datetime import datetime

# Create/connect to database
conn = sqlite3.connect(':memory:')  # Use ':memory:' for in-memory DB, or 'notes.db' for file
cursor = conn.cursor()

# Create notes table
cursor.execute('''
    CREATE TABLE notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
''')

# Insert 3 sample notes
sample_notes = [
    ('Meeting Notes', 'Discussed Q2 roadmap and timeline', datetime.now().isoformat()),
    ('Project Ideas', 'Build a task manager app with sync', datetime.now().isoformat()),
    ('Reminder', 'Update documentation before release', datetime.now().isoformat())
]

cursor.executemany(
    'INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)',
    sample_notes
)

conn.commit()

# Query all notes
cursor.execute('SELECT * FROM notes')
notes = cursor.fetchall()

# Print results
print("All Notes:")
print("-" * 60)
for note in notes:
    print(f"ID: {note[0]}")
    print(f"Title: {note[1]}")
    print(f"Body: {note[2]}")
    print(f"Created: {note[3]}")
    print("-" * 60)

conn.close()
```

**To run it:**

```bash
python3 script.py
```

The script will output all 3 notes with their IDs, titles, bodies, and creation timestamps. If you want to persist the database to disk instead of memory, change `':memory:'` to `'notes.db'`.
