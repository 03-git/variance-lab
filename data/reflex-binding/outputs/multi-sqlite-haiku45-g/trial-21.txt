Here's a Python script that creates a SQLite database with notes and displays them:

```python
import sqlite3
from datetime import datetime

# Create in-memory database (or use 'notes.db' for a file)
conn = sqlite3.connect(':memory:')
cursor = conn.cursor()

# Create notes table
cursor.execute('''
    CREATE TABLE notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
''')

# Insert sample notes
sample_notes = [
    ('Shopping List', 'Milk, eggs, bread, butter', datetime.now().isoformat()),
    ('Meeting Notes', 'Discuss Q2 roadmap, review metrics', datetime.now().isoformat()),
    ('Project Ideas', 'Build CLI tool, write docs, refactor auth', datetime.now().isoformat()),
]

cursor.executemany(
    'INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)',
    sample_notes
)
conn.commit()

# Query and display all notes
cursor.execute('SELECT * FROM notes')
notes = cursor.fetchall()

print("All Notes:")
print("-" * 60)
for note in notes:
    note_id, title, body, created_at = note
    print(f"ID: {note_id}")
    print(f"Title: {title}")
    print(f"Body: {body}")
    print(f"Created: {created_at}")
    print("-" * 60)

conn.close()
```

**To run it:**

```bash
python3 script.py
```

This script:
- Creates an in-memory SQLite database (change `':memory:'` to `'notes.db'` to persist to a file)
- Creates a `notes` table with the schema you specified
- Inserts 3 sample notes with timestamps
- Queries all notes and prints them formatted

If you want to use a persistent file instead of in-memory, just change line 4 to `conn = sqlite3.connect('notes.db')` and the database will be saved as `notes.db`.
