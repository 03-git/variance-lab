Here's a Python script that does exactly that:

```python
import sqlite3
from datetime import datetime

# Create an in-memory database (or use a file with sqlite3.connect('notes.db'))
conn = sqlite3.connect(':memory:')
cursor = conn.cursor()

# Create the notes table
cursor.execute('''
    CREATE TABLE notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
''')

# Insert 3 sample notes
now = datetime.now().isoformat()
cursor.execute('INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)',
               ('First Note', 'This is my first note content.', now))
cursor.execute('INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)',
               ('Second Note', 'Here is some more content for the second note.', now))
cursor.execute('INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)',
               ('Third Note', 'And finally, a third note with different information.', now))

conn.commit()

# Query and print all notes
cursor.execute('SELECT * FROM notes')
notes = cursor.fetchall()

print("All Notes:")
print("-" * 60)
for note in notes:
    note_id, title, body, created_at = note
    print(f"ID: {note_id}")
    print(f"Title: {title}")
    print(f"Body: {body}")
    print(f"Created: {created_at}")
    print("-" * 60)

conn.close()
```

**How to run it:**

1. Save the code to a file (e.g., `notes.py`)
2. Run it with: `python notes.py`

The script will output all 3 notes with their ID, title, body, and creation timestamp.

If you want to persist the database to a file instead of using in-memory storage, change the first line after imports to:
```python
conn = sqlite3.connect('notes.db')
```
