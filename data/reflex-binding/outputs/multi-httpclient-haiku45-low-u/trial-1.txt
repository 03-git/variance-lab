Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print(f"\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        if len(response.text) > 500:
            print("...")
            
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
python script.py https://httpbin.org/redirect/2
```

The script uses the `requests` library which handles redirects automatically with `allow_redirects=True`. It prints the final status code (after all redirects), all response headers, and the first 500 characters of the response body. The timeout is set to 10 seconds to avoid hanging on unresponsive servers.
