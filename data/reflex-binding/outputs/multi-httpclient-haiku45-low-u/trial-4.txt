Here's a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    """Fetch a URL and print status code, headers, and body preview."""
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}\n")
        
        print("Response Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        print(response.text[:500])
        
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

**Features:**
- `allow_redirects=True` — automatically follows redirects
- `timeout=10` — 10-second timeout to avoid hanging
- Prints status code, all headers, and first 500 characters of the response body
- Error handling for network issues
