Here's a Python script that does what you're asking for:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print("\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        
        if len(response.text) > 500:
            print(f"\n... (truncated, total body size: {len(response.text)} characters)")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
python script.py https://httpbin.org/redirect-to?url=https://example.com
```

**What it does:**
- Takes a URL from command-line argument
- Makes a GET request with `allow_redirects=True` (follows redirects automatically)
- Prints the final status code, all response headers, and the first 500 characters of the response body
- Includes a 10-second timeout and basic error handling
- Shows a message if the body is longer than 500 characters
