I'll create a Python script that does exactly that. Here's the code:

```python
import sys
import requests

def fetch_url(url):
    """Fetch a URL and print status code, headers, and body preview."""
    try:
        # Make GET request following redirects (default behavior)
        response = requests.get(url, timeout=10)
        
        # Print status code
        print(f"Status Code: {response.status_code}")
        print()
        
        # Print response headers
        print("Response Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        print()
        
        # Print first 500 characters of body
        body_preview = response.text[:500]
        print("Response Body (first 500 characters):")
        print(body_preview)
        if len(response.text) > 500:
            print(f"\n... (truncated, total length: {len(response.text)} characters)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>")
        print("Example: python script.py https://example.com")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

1. First, make sure you have the `requests` library installed:
```bash
pip install requests
```

2. Save the script to a file (e.g., `fetch_url.py`) and run it:
```bash
python fetch_url.py https://example.com
```

3. Or run it directly from the command line:
```bash
python fetch_url.py https://www.google.com
```

The script will automatically follow redirects (that's the default behavior of `requests.get()`), print the final status code, all response headers, and the first 500 characters of the response body.
