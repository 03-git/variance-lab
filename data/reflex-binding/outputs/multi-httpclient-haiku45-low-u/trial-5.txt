I'll show you a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print(f"\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        if len(response.text) > 500:
            print("... (truncated)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

Or with a URL that redirects:
```bash
python script.py https://bit.ly/some-short-url
```

**What it does:**
- Takes a URL as the only command-line argument
- Makes a GET request with `allow_redirects=True` to follow redirects automatically
- Prints the final status code (after all redirects)
- Prints all response headers
- Prints the first 500 characters of the response body
- Handles errors gracefully and includes a 10-second timeout

You'll need the `requests` library installed:
