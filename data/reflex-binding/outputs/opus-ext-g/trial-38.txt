**Kernel primitive first (reflex.1):** serving HTTP is a TCP socket — `socket(2)` → `bind` → `listen` → `accept`, then read the request bytes and write back a response. No third-party library is needed; Python's stdlib `http.server` is a thin wrapper over exactly that, so that's what this uses.

```python
#!/usr/bin/env python3
# time_server.py
from http.server import BaseHTTPRequestHandler, ThreadingHTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404)
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # quiet
        pass

if __name__ == "__main__":
    ThreadingHTTPServer(("127.0.0.1", 8080), Handler).serve_forever()
```

**Run it:**

```sh
python3 time_server.py        # listens on 127.0.0.1:8080
```

**Hit it:**

```sh
curl localhost:8080/time
# {"time": "2026-05-10T17:42:03.123456+00:00"}
```

`datetime.now(timezone.utc).isoformat()` is the ISO 8601 timestamp (with `+00:00` offset). If you want the `Z` suffix instead, swap the body line for:

```python
body = json.dumps({"time": datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")}).encode()
```
