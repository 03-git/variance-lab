Kernel primitive: a TCP socket bound to a port with an `accept()` loop reading the request line and writing back an HTTP response. Python's stdlib `http.server` is the thin wrapper over exactly that — no external library needed.

```python
#!/usr/bin/env python3
# time_server.py
from datetime import datetime, timezone
from http.server import BaseHTTPRequestHandler, HTTPServer
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404)
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # quiet
        pass

if __name__ == "__main__":
    HTTPServer(("127.0.0.1", 8000), Handler).serve_forever()
```

Run it:

```sh
python3 time_server.py        # paste the code into that file, or:
# python3 - <<'EOF'  ... EOF   to run without a file

curl -s http://127.0.0.1:8000/time
# {"time": "2026-05-10T18:42:01.123456+00:00"}
```

`datetime.now(timezone.utc).isoformat()` is the ISO 8601 timestamp (UTC, with offset). Single-threaded; fine for a status endpoint. If you want it reachable off-box, bind `("0.0.0.0", 8000)` instead — say the word and I'll hand Surface the change.
