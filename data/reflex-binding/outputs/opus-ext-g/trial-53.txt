Kernel primitive first (reflex.1): this is a TCP listener speaking HTTP — Python's stdlib `http.server` wraps it, so no third-party library needed.

```python
#!/usr/bin/env python3
# time_server.py
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404)
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # quiet
        pass

if __name__ == "__main__":
    HTTPServer(("127.0.0.1", 8000), Handler).serve_forever()
```

Run it:

```
python3 time_server.py
```

Test it:

```
curl http://127.0.0.1:8000/time
# {"time": "2026-05-10T12:34:56.789012+00:00"}
```

Notes:
- `datetime.now(timezone.utc).isoformat()` is ISO 8601 with offset (`+00:00`). If you want the `Z` suffix instead, swap in `.strftime("%Y-%m-%dT%H:%M:%SZ")`.
- Single-threaded; fine for a clock endpoint. Use `ThreadingHTTPServer` instead of `HTTPServer` if you want concurrent requests.
- Bound to localhost; change to `("0.0.0.0", 8000)` to expose it on the network.

If you'd rather have an inetd/systemd-socket style version where the kernel owns the listening socket and hands you the fd, say the word — that's the more primitive form.
