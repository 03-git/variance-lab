```python
import unittest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):

    def test_freezing_point(self):
        self.assertEqual(celsius_to_fahrenheit(0), 32)

    def test_boiling_point(self):
        self.assertEqual(celsius_to_fahrenheit(100), 212)

    def test_equal_point(self):
        # -40 is the point where Celsius and Fahrenheit are equal
        self.assertEqual(celsius_to_fahrenheit(-40), -40)

    def test_non_numeric_input(self):
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("hot")

if __name__ == "__main__":
    unittest.main()
```

**To run it**, save the code to a file (e.g. `test_temp.py`) and run:

```bash
python -m unittest test_temp.py -v
```

Or since the file has the `__main__` block:

```bash
python test_temp.py
```

**Expected output:**
```
test_boiling_point ... ok
test_equal_point ... ok
test_freezing_point ... ok
test_non_numeric_input ... ok

----------------------------------------------------------------------
Ran 4 tests in 0.001s

OK
```

Note: the non-numeric test expects a `TypeError`, which Python raises naturally when you try to multiply a string by a float — no extra error handling needed in the function itself.
