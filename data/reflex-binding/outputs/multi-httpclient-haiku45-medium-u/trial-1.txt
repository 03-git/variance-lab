I'll show you a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        print(response.text[:500])
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

1. Save it to a file (e.g., `fetch_url.py`)
2. Make sure you have `requests` installed: `pip install requests`
3. Run it with: `python fetch_url.py https://example.com`

**Example output:**
```
Status Code: 200

Response Headers:
  Content-Type: text/html; charset=UTF-8
  Content-Length: 1234
  ...

First 500 characters of body:
<!doctype html><html><head><title>Example Domain</title>...
```

The script uses `requests.get()` with `allow_redirects=True` (the default) to follow redirects automatically, and displays the final response's status code, headers, and first 500 characters of the body.
