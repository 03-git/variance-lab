I'll show you a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    try:
        # Make GET request following redirects (default behavior)
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        # Print status code
        print(f"Status Code: {response.status_code}")
        
        # Print response headers
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        # Print first 500 characters of body
        print("\nResponse Body (first 500 characters):")
        body = response.text[:500]
        print(body)
        if len(response.text) > 500:
            print("... (truncated)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

1. **Save the script** (e.g., as `fetch_url.py`)
2. **Install requests** (if you don't have it):
   ```bash
   pip install requests
   ```
3. **Run it**:
   ```bash
   python fetch_url.py https://example.com
   ```
