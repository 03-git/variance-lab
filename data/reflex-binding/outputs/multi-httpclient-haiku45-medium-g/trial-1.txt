Here's a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print("\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

Or with a URL that redirects:

```bash
python script.py https://bit.ly/somelink
```

**What it does:**
- Takes a URL from the command line
- Makes a GET request with `allow_redirects=True` to follow any redirects
- Prints the final status code
- Prints all response headers
- Prints the first 500 characters of the response body
- Handles errors gracefully with a timeout of 10 seconds

Make sure you have the `requests` library installed:
