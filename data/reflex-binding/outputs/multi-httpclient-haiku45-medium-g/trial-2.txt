Here's a Python script that does what you need:

```python
#!/usr/bin/env python3
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for header, value in response.headers.items():
        print(f"  {header}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python3 script.py "https://example.com"
```

The script:
- Takes a URL as the first command-line argument
- Makes a GET request with `allow_redirects=True` to follow redirects automatically
- Prints the status code of the final response
- Prints all response headers
- Prints the first 500 characters of the response body
- Handles errors gracefully with a 10-second timeout

If you don't have `requests` installed, install it with `pip install requests`.
