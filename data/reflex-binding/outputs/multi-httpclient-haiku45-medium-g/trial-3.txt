Here's a Python script that does exactly that:

```python
#!/usr/bin/env python3
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nHeaders:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    body = response.text[:500]
    print(body)
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**To run it:**

```bash
python3 script.py "https://example.com"
```

The script:
- Takes a URL as the only command-line argument
- Uses `requests.get()` with `allow_redirects=True` (default behavior) to follow redirects
- Prints the final status code after all redirects
- Prints all response headers
- Prints the first 500 characters of the response body
- Has a 10-second timeout and basic error handling for network issues

Note: You'll need the `requests` library installed (`pip install requests`).
