I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

Here's the code using Flask (a lightweight web framework):

```python
from flask import Flask
import json
from datetime import datetime, timezone

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    current_time = datetime.now(timezone.utc).isoformat()
    return {'time': current_time}

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

1. Install Flask if you don't have it:
   ```bash
   pip install flask
   ```

2. Run the script:
   ```bash
   python script.py
   ```

3. Test the endpoint:
   ```bash
   curl http://localhost:5000/time
   ```

You'll get a response like:
```json
{"time":"2026-05-10T14:23:45.123456+00:00"}
```

**Alternative using the built-in `http.server`** (no external dependencies):

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
import json
from datetime import datetime, timezone

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            current_time = datetime.now(timezone.utc).isoformat()
            response = json.dumps({'time': current_time})
            
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            self.wfile.write(response.encode())
        else:
            self.send_response(404)
            self.end_headers()

if __name__ == '__main__':
    server = HTTPServer(('localhost', 5000), TimeHandler)
    print('Server running on http://localhost:5000')
    server.serve_forever()
```

Run with `python script.py`, then test with `curl http://localhost:5000/time`.

The Flask version is simpler; the `http.server` version has zero dependencies.
