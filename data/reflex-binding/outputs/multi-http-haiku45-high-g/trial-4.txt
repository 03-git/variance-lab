I'll show you a simple HTTP endpoint in Python using Flask that returns the current time as JSON.

```python
from flask import Flask
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return {
        'time': datetime.utcnow().isoformat() + 'Z'
    }

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

```bash
pip install flask
python your_script.py
```

Then visit `http://localhost:5000/time` in your browser or curl it:

```bash
curl http://localhost:5000/time
```

You'll get output like:
```json
{"time": "2026-05-10T14:23:45.123456Z"}
```

If you prefer a different framework, here's the same thing with **http.server** (no dependencies):

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            response = {
                'time': datetime.utcnow().isoformat() + 'Z'
            }
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, format, *args):
        pass

if __name__ == '__main__':
    server = HTTPServer(('localhost', 5000), TimeHandler)
    print('Server running at http://localhost:5000/time')
    server.serve_forever()
```

**To run the built-in version:**

```bash
python your_script.py
```

No dependencies needed. Pick whichever fits your use case — Flask is simpler for larger projects, the built-in version is lighter if you just need a single endpoint.
