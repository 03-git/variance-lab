I'll show you a Python script that fetches a URL and extracts hyperlinks from the HTML.

```python
import requests
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self.current_text = []
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            attrs_dict = dict(attrs)
            href = attrs_dict.get('href', '')
            self.current_text = []
    
    def handle_data(self, data):
        self.current_text.append(data.strip())
    
    def handle_endtag(self, tag):
        if tag == 'a':
            link_text = ' '.join(self.current_text).strip()
            if self.links and self.links[-1][0]:  # if we're in an anchor tag
                self.links[-1] = (self.links[-1][0], link_text)
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            attrs_dict = dict(attrs)
            href = attrs_dict.get('href', '')
            self.links.append((href, ''))
            self.current_text = []

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        parser = LinkExtractor()
        parser.feed(response.text)
        
        return parser.links
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return []

if __name__ == '__main__':
    url = 'https://example.com'
    links = extract_links(url)
    
    print(f"Found {len(links)} links:\n")
    for href, text in links:
        print(f"URL: {href}")
        print(f"Text: {text if text else '(no text)'}")
        print()
```

**How to run it:**

1. Install the required library:
```bash
pip install requests
```

2. Run the script:
```bash
python script.py
```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses Python's built-in `HTMLParser` to extract all `<a>` tags
- Prints each link's href and the link text (the text between the opening and closing tags)
- Handles errors gracefully if the URL can't be fetched
