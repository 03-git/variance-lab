I'll show you a Python script that fetches a URL and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            
            # Handle relative URLs
            if href:
                full_url = urljoin(url, href)
                print(f"Text: {text or '(no text)'}")
                print(f"URL:  {full_url}\n")
            else:
                print(f"Text: {text or '(no text)'}")
                print(f"URL:  (no href)\n")
    
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the URL using `requests`
- Parses the HTML with `BeautifulSoup`
- Extracts all `<a>` tags
- Prints the link text and href for each link
- Converts relative URLs to absolute URLs using `urljoin`
- Handles errors gracefully

You can modify the URL by changing the argument in the last line, or make it accept command-line arguments if needed.
