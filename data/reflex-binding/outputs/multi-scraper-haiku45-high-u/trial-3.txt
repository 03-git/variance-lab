I'll show you a Python script that fetches a URL and extracts hyperlinks:

```python
import requests
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self.current_text = []
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            href = dict(attrs).get('href', '')
            self.links.append({'href': href, 'text': ''})
            self.current_text = []
    
    def handle_data(self, data):
        if self.links:
            self.current_text.append(data.strip())
    
    def handle_endtag(self, tag):
        if tag == 'a' and self.links:
            self.links[-1]['text'] = ' '.join(self.current_text)
            self.current_text = []
    
    def get_links(self):
        return self.links

# Fetch and parse
url = 'https://example.com'
response = requests.get(url, timeout=10)
response.raise_for_status()

parser = LinkExtractor()
parser.feed(response.text)

# Print results
for link in parser.get_links():
    href = link['href']
    text = link['text'] or '(no text)'
    print(f"Text: {text}")
    print(f"Href: {href}")
    print()
```

**How to run it:**

1. Install the required library (if you don't have it):
   ```bash
   pip install requests
   ```

2. Save the script to a file (e.g., `fetch_links.py`) and run:
   ```bash
   python fetch_links.py
   ```

Or run it directly in the Python REPL by pasting the code.

**What it does:**
- Uses `requests` to fetch the HTML
- Uses Python's built-in `HTMLParser` to parse anchor tags
- Extracts both the `href` attribute and the link text
- Prints each link's text and URL

Note: `requests` is the only external dependency. The HTML parsing uses Python's standard library.
