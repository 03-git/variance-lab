I'll show you a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from html.parser import HTMLParser
from urllib.parse import urljoin

class LinkExtractor(HTMLParser):
    def __init__(self, base_url):
        super().__init__()
        self.base_url = base_url
        self.links = []
        self.current_text = []
        self.in_link = False
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            self.in_link = True
            self.current_text = []
            for attr, value in attrs:
                if attr == 'href':
                    self.href = value
    
    def handle_data(self, data):
        if self.in_link:
            self.current_text.append(data.strip())
    
    def handle_endtag(self, tag):
        if tag == 'a' and self.in_link:
            self.in_link = False
            link_text = ' '.join(self.current_text)
            # Convert relative URLs to absolute
            absolute_url = urljoin(self.base_url, self.href)
            self.links.append((absolute_url, link_text))

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        parser = LinkExtractor(url)
        parser.feed(response.text)
        
        return parser.links
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return []

if __name__ == "__main__":
    url = "https://example.com"
    links = extract_links(url)
    
    if links:
        print(f"Found {len(links)} links:\n")
        for href, text in links:
            print(f"URL: {href}")
            print(f"Text: {text if text else '(no text)'}")
            print()
    else:
        print("No links found or error occurred.")
```

**How to run it:**

1. **Install the required package** (if not already installed):
   ```bash
   pip install requests
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from the specified URL
- Uses `HTMLParser` to parse the HTML and extract all `<a>` tags
- Captures both the `href` attribute and the link text
- Converts relative URLs to absolute URLs using `urljoin`
- Prints each URL and its corresponding link text

**Output example:**
```
Found 3 links:

URL: https://www.iana.org/domains/example
Text: More information...

URL: https://www.iana.org/
Text: 

...
```

You can modify the `url` variable at the bottom to scrape any website. Note that example.com has very few links, so you might want to test with a more complex site.
