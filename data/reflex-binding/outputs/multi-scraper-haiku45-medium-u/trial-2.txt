I'll show you a Python script that fetches a website and extracts all hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks from it."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise an error for bad status codes
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        
        # Extract and print href and link text
        for link in links:
            href = link.get('href')
            text = link.get_text(strip=True)
            
            # Convert relative URLs to absolute
            if href:
                absolute_href = urljoin(url, href)
                print(f"Text: {text if text else '(no text)'}")
                print(f"URL:  {absolute_href}\n")
            else:
                print(f"Text: {text if text else '(no text)'}")
                print(f"URL:  (no href)\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the URL: {e}")

if __name__ == "__main__":
    url = "https://example.com"
    fetch_and_extract_links(url)
```

## How to run it:

**1. Install the required libraries:**
```bash
pip install requests beautifulsoup4
```

**2. Run the script:**
```bash
python script.py
```

## What it does:
- Fetches the HTML from the provided URL using `requests`
- Parses the HTML using BeautifulSoup
- Finds all `<a>` tags and extracts their `href` attributes and text content
- Converts relative URLs to absolute URLs using `urljoin`
- Handles errors gracefully (network issues, bad status codes)

## Optional: To test with a different URL
