I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

**Using Python's built-in `http.server`:**

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            response = {'time': datetime.utcnow().isoformat() + 'Z'}
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()
    
    def log_message(self, format, *args):
        pass  # Suppress logs

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print('Server running at http://localhost:8000/time')
    server.serve_forever()
```

**To run it:**
```bash
python3 server.py
```

Then in another terminal:
```bash
curl http://localhost:8000/time
```

You'll get: `{"time": "2026-05-10T14:32:45.123456Z"}`

---

**Alternative using Flask (if you have it installed):**

```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time')
def get_time():
    return jsonify({'time': datetime.utcnow().isoformat() + 'Z'})

if __name__ == '__main__':
    app.run(port=8000)
```

Run with `python3 server.py` and test the same way. Flask is lighter weight for a single endpoint, but the first example needs no dependencies.
