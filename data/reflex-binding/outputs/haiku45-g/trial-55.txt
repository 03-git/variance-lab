I'll show you a simple HTTP endpoint that returns the current time in JSON format. Here are a few options:

## Node.js + Express

```javascript
const express = require('express');
const app = express();

app.get('/time', (req, res) => {
  res.json({ time: new Date().toISOString() });
});

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

Run with: `node app.js` (after `npm install express`)

## Python + Flask

```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return jsonify({'time': datetime.utcnow().isoformat() + 'Z'})

if __name__ == '__main__':
    app.run(port=3000)
```

Run with: `python app.py` (after `pip install flask`)

## Python + http.server (no dependencies)

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            response = {'time': datetime.utcnow().isoformat() + 'Z'}
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()

HTTPServer(('0.0.0.0', 3000), TimeHandler).serve_forever()
```

Run with: `python app.py` (no dependencies needed)

Test with: `curl http://localhost:3000/time`
