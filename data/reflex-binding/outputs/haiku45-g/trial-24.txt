I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

```python
from flask import Flask
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return {
        'time': datetime.now().isoformat()
    }

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

```bash
pip install flask
python app.py
```

Then access it at `http://localhost:5000/time` and you'll get a response like:

```json
{
  "time": "2026-05-10T14:23:45.123456"
}
```

**Alternative using plain Python (no dependencies):**

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            response = {'time': datetime.now().isoformat()}
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print('Server running on http://localhost:8000/time')
    server.serve_forever()
```

**To run the plain Python version:**

```bash
python app.py
```

Then access it at `http://localhost:8000/time`. This version requires no external dependencies.
