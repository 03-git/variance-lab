I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            response = {
                'time': datetime.utcnow().isoformat() + 'Z'
            }
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()
    
    def log_message(self, format, *args):
        pass  # Suppress request logging

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print('Server running at http://localhost:8000')
    server.serve_forever()
```

**To run it:**
```bash
python3 /path/to/script.py
```

Then test it:
```bash
curl http://localhost:8000/time
# Returns: {"time": "2026-05-10T14:32:45.123456Z"}
```

**Alternative using Flask** (if you prefer):
```python
from flask import Flask
from datetime import datetime

app = Flask(__name__)

@app.route('/time')
def get_time():
    return {'time': datetime.utcnow().isoformat() + 'Z'}

if __name__ == '__main__':
    app.run(host='localhost', port=8000)
```

Run with: `flask run --port 8000`

The built-in server is lighter-weight; Flask is more feature-rich if you need to expand it later.
