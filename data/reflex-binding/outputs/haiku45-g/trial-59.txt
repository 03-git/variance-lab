I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

**Using Flask** (lightweight and straightforward):

```python
from flask import Flask
from datetime import datetime, timezone

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    iso_time = datetime.now(timezone.utc).isoformat()
    return {'time': iso_time}

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**Using FastAPI** (modern async alternative):

```python
from fastapi import FastAPI
from datetime import datetime, timezone

app = FastAPI()

@app.get('/time')
def get_time():
    iso_time = datetime.now(timezone.utc).isoformat()
    return {'time': iso_time}
```

**Using stdlib only** (http.server, no dependencies):

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            iso_time = datetime.now(timezone.utc).isoformat()
            response = json.dumps({'time': iso_time})
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(response.encode())
        else:
            self.send_response(404)
            self.end_headers()
    
    def log_message(self, format, *args):
        pass  # silence request logs

if __name__ == '__main__':
    server = HTTPServer(('localhost', 5000), TimeHandler)
    print('Server running on http://localhost:5000')
    server.serve_forever()
```

**How to run:**

Flask: `pip install flask && python app.py`
FastAPI: `pip install fastapi uvicorn && uvicorn app:app --reload`
Stdlib: `python app.py` (no install needed)

**Test it:**
```bash
curl http://localhost:5000/time
# Returns: {"time":"2026-05-10T14:32:45.123456+00:00"}
```

Pick whichever fits your needs — Flask/FastAPI if you want ecosystem support, stdlib if you want zero dependencies.
