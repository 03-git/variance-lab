I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            response = {
                'time': datetime.utcnow().isoformat() + 'Z'
            }
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, format, *args):
        pass  # Suppress request logging

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print('Server running at http://localhost:8000/time')
    server.serve_forever()
```

**To run it:**
```bash
python3 endpoint.py
```

Then test it:
```bash
curl http://localhost:8000/time
```

You'll get output like:
```json
{"time": "2026-05-10T14:32:45.123456Z"}
```

**Using Flask (simpler alternative):**
```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    return jsonify({'time': datetime.utcnow().isoformat() + 'Z'})

if __name__ == '__main__':
    app.run(port=8000)
```

Run with: `python3 endpoint.py` (after `pip install flask`)

Both return ISO 8601 timestamps. The first uses only stdlib; the second is more concise if Flask is available.
