I'll show you a simple HTTP endpoint in Python that returns the current time as JSON.

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            self.send_response(200)
            self.send_header('Content-type', 'application/json')
            self.end_headers()
            response = {"time": datetime.utcnow().isoformat() + "Z"}
            self.wfile.write(json.dumps(response).encode())
        else:
            self.send_response(404)
            self.end_headers()
    
    def log_message(self, format, *args):
        pass  # Suppress request logging

if __name__ == '__main__':
    server = HTTPServer(('localhost', 8000), TimeHandler)
    print("Server running at http://localhost:8000/time")
    server.serve_forever()
```

**To run it:**

```bash
python3 endpoint.py
```

Then test it with:

```bash
curl http://localhost:8000/time
```

This will return something like: `{"time": "2026-05-10T14:32:45.123456Z"}`

The endpoint listens on `localhost:8000`, responds to `GET /time` with the current UTC time in ISO 8601 format, and returns a 404 for any other path.
