Here's a minimal async TCP echo server using `asyncio`.

```python
import asyncio

async def handle_client(reader: asyncio.StreamReader, writer: asyncio.StreamWriter) -> None:
    addr = writer.get_extra_info("peername")
    print(f"connected: {addr}")
    try:
        while True:
            line = await reader.readline()
            if not line:
                break
            writer.write(line)
            await writer.drain()
    except ConnectionResetError:
        pass
    finally:
        print(f"disconnected: {addr}")
        writer.close()
        await writer.wait_closed()

async def main(host: str = "127.0.0.1", port: int = 8888) -> None:
    server = await asyncio.start_server(handle_client, host, port)
    addrs = ", ".join(str(s.getsockname()) for s in server.sockets)
    print(f"listening on {addrs}")
    async with server:
        await server.serve_forever()

if __name__ == "__main__":
    try:
        asyncio.run(main())
    except KeyboardInterrupt:
        pass
```

## How it works

- `asyncio.start_server` accepts connections and spawns `handle_client` as a task per client, so many clients run concurrently on a single thread.
- `reader.readline()` awaits until a `\n`-terminated line arrives (or EOF on disconnect).
- `writer.drain()` applies backpressure so a slow client can't blow up memory.

## Run it

Save as `echo_server.py`, then:

```
python3 echo_server.py
```

## Test it

In another terminal, connect with `nc` (or `telnet`) — open several at once to confirm concurrency:

```
nc 127.0.0.1 8888
hello
hello
```

Or a quick Python client:

```python
import asyncio

async def client(msg: str) -> None:
    r, w = await asyncio.open_connection("127.0.0.1", 8888)
    w.write((msg + "\n").encode())
    await w.drain()
    print(await r.readline())
    w.close()
    await w.wait_closed()

async def main() -> None:
    await asyncio.gather(*(client(f"msg-{i}") for i in range(10)))

asyncio.run(main())
```

That fires 10 concurrent clients; each gets its own echo back.
