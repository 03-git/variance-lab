I'll build a simple REST API endpoint for you using Python with Flask. Here's the complete code:

```python
from flask import Flask, request, jsonify
from datetime import datetime

app = Flask(__name__)

def validate_user(data):
    """Validate user input and return (is_valid, user_dict_or_errors)"""
    errors = {}
    
    # Validate name
    if 'name' not in data:
        errors['name'] = 'name is required'
    elif not isinstance(data['name'], str) or not data['name'].strip():
        errors['name'] = 'name must be a non-empty string'
    
    # Validate email
    if 'email' not in data:
        errors['email'] = 'email is required'
    elif not isinstance(data['email'], str) or '@' not in data['email']:
        errors['email'] = 'email must be a string containing @'
    
    # Validate age
    if 'age' not in data:
        errors['age'] = 'age is required'
    elif not isinstance(data['age'], int) or data['age'] < 1 or data['age'] > 150:
        errors['age'] = 'age must be an integer between 1 and 150'
    
    if errors:
        return False, errors
    
    user = {
        'name': data['name'].strip(),
        'email': data['email'].strip(),
        'age': data['age'],
        'created_at': datetime.utcnow().isoformat()
    }
    return True, user

@app.route('/users', methods=['POST'])
def create_user():
    try:
        data = request.get_json()
        if data is None:
            return jsonify({'error': 'request body must be valid JSON'}), 400
    except Exception as e:
        return jsonify({'error': 'invalid JSON'}), 400
    
    is_valid, result = validate_user(data)
    
    if not is_valid:
        return jsonify({'errors': result}), 400
    
    return jsonify(result), 201

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

**To run it:**

```bash
# Install Flask if you don't have it
pip install flask

# Save the code to a file (e.g., app.py) and run
python app.py
```

The server will start on `http://localhost:5000`

**Test it with curl:**

```bash
# Valid request
curl -X POST http://localhost:5000/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Alice", "email": "alice@example.com", "age": 30}'

# Missing name
curl -X POST http://localhost:5000/users \
  -H "Content-Type: application/json" \
  -d '{"email": "bob@example.com", "age": 25}'

# Invalid email (no @)
curl -X POST http://localhost:5000/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Charlie", "email": "charlie", "age": 35}'

# Invalid age (out of range)
curl -X POST http://localhost:5000/users \
  -H "Content-Type: application/json" \
  -d '{"name": "David", "email": "david@example.com", "age": 200}'
```

The endpoint validates all inputs, returns 400 with detailed error messages for invalid data, and 201 with the user object (including a timestamp) on success.
