```python
import sys
import urllib.request

url = sys.argv[1]
resp = urllib.request.urlopen(url)
print(f"Status: {resp.status}")
print("\nHeaders:")
for k, v in resp.getheaders():
    print(f"  {k}: {v}")
body = resp.read(500).decode("utf-8", errors="replace")
print(f"\nBody (first 500 chars):\n{body}")
```

Run it with:

```
python script.py https://example.com
```

`urllib.request.urlopen` follows redirects by default. No third-party dependencies needed.
