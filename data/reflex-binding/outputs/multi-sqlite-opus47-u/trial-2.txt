```python
import sqlite3
from datetime import datetime, timezone

conn = sqlite3.connect("notes.db")
cur = conn.cursor()

cur.execute("""
    CREATE TABLE IF NOT EXISTS notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
""")

now = datetime.now(timezone.utc).isoformat()
samples = [
    ("Groceries", "milk, eggs, bread", now),
    ("Ideas", "write a SQLite demo", now),
    ("Reminder", "call the dentist", now),
]
cur.executemany(
    "INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)",
    samples,
)
conn.commit()

for row in cur.execute("SELECT id, title, body, created_at FROM notes"):
    print(row)

conn.close()
```

Run it:

```
python3 notes.py
```

Notes:
- No external dependencies — `sqlite3` ships with Python.
- It writes `notes.db` in the current directory; delete it between runs (or the table will accumulate duplicates) since `CREATE TABLE IF NOT EXISTS` keeps existing rows.
