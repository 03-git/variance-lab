Here's a minimal implementation using only the Python standard library — no dependencies, no files needed beyond this one script.

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404, "Not Found")
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # quiet logging; remove to see requests
        pass

if __name__ == "__main__":
    HTTPServer(("", 8000), TimeHandler).serve_forever()
```

## How to run it

1. Save the snippet as `time_server.py` (or paste it into a file).
2. Run it (Python 3.7+):
   ```sh
   python3 time_server.py
   ```
3. In another terminal:
   ```sh
   curl http://localhost:8000/time
   ```
   Response:
   ```json
   {"time": "2026-05-10T14:23:51.482913+00:00"}
   ```

Stop the server with `Ctrl+C`.

Notes:
- The timestamp is UTC (`+00:00`); drop `timezone.utc` from `datetime.now()` if you want local time without an offset, or keep it for unambiguous ISO 8601.
- Any path other than `/time` returns 404.
- Want a Node.js or Flask/FastAPI version instead? Say the word.
