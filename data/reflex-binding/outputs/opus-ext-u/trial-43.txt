Here's a minimal implementation using only the Python standard library — no dependencies, no files needed beyond this one script.

```python
# time_server.py
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404, "Not Found")
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # quiet logging; remove to see requests
        pass

if __name__ == "__main__":
    HTTPServer(("127.0.0.1", 8000), TimeHandler).serve_forever()
```

Run it:

```sh
python3 time_server.py
```

Test it:

```sh
curl http://127.0.0.1:8000/time
# {"time": "2026-05-10T17:23:45.123456+00:00"}
```

---

If you'd rather have it as a one-liner-ish script without saving a file at all:

```sh
python3 -c '
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json
class H(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time": return self.send_error(404)
        b = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200); self.send_header("Content-Type","application/json"); self.end_headers(); self.wfile.write(b)
HTTPServer(("127.0.0.1",8000), H).serve_forever()
'
```

Notes:
- `datetime.now(timezone.utc).isoformat()` produces a valid ISO 8601 string with offset (e.g. `...+00:00`). If you want the `Z` suffix instead, swap in `datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")`.
- It binds to `127.0.0.1` (localhost only). Use `"0.0.0.0"` to accept external connections.
- `BaseHTTPRequestHandler` is single-threaded; fine for testing. For concurrency use `ThreadingHTTPServer` instead of `HTTPServer`.
