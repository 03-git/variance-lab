Here's a minimal implementation using Python's standard library — no dependencies, no files needed beyond what you choose to create.

```python
# server.py
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404, "Not Found")
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def log_message(self, *args):  # silence default request logging (optional)
        pass

if __name__ == "__main__":
    HTTPServer(("127.0.0.1", 8000), TimeHandler).serve_forever()
```

## Run it

```sh
python3 server.py
```

Then in another terminal:

```sh
curl http://127.0.0.1:8000/time
```

Response:

```json
{"time": "2026-05-10T14:23:01.123456+00:00"}
```

## One-liner alternative (no script at all)

```sh
python3 -c 'from http.server import *; from datetime import *; import json; H=type("H",(BaseHTTPRequestHandler,),{"do_GET":lambda s:(s.send_response(200),s.send_header("Content-Type","application/json"),s.end_headers(),s.wfile.write(json.dumps({"time":datetime.now(timezone.utc).isoformat()}).encode()))}); HTTPServer(("127.0.0.1",8000),H).serve_forever()'
```

(The one-liner ignores the path; the full script enforces `GET /time` and 404s everything else.)
