Here's a minimal implementation using only Python's standard library — no dependencies, no files needed if you just paste it into a `python3` session or save it yourself.

```python
from http.server import BaseHTTPRequestHandler, HTTPServer
from datetime import datetime, timezone
import json

class TimeHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path != "/time":
            self.send_error(404, "Not Found")
            return
        body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

if __name__ == "__main__":
    HTTPServer(("127.0.0.1", 8000), TimeHandler).serve_forever()
```

## How to run it

1. Save the code as `time_server.py`.
2. Start it (Python 3.x, no install step needed):
   ```sh
   python3 time_server.py
   ```
3. In another terminal, hit the endpoint:
   ```sh
   curl http://127.0.0.1:8000/time
   ```
   Response:
   ```json
   {"time": "2026-05-10T17:42:13.512874+00:00"}
   ```

Stop the server with `Ctrl+C`.
