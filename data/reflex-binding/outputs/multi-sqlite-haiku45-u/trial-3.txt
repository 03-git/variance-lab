Here's a Python script that does what you need:

```python
import sqlite3
from datetime import datetime

# Create an in-memory database (or use ':memory:' for temporary DB)
conn = sqlite3.connect(':memory:')
cursor = conn.cursor()

# Create the notes table
cursor.execute('''
    CREATE TABLE notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
''')

# Insert 3 sample notes
now = datetime.now().isoformat()
cursor.execute('''
    INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)
''', ('First Note', 'This is my first note content.', now))

cursor.execute('''
    INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)
''', ('Second Note', 'Here is the second note with more details.', now))

cursor.execute('''
    INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)
''', ('Third Note', 'The third note to complete the set.', now))

conn.commit()

# Query all notes
cursor.execute('SELECT * FROM notes')
notes = cursor.fetchall()

# Print the notes
print("All Notes:")
print("-" * 50)
for note in notes:
    print(f"ID: {note[0]}")
    print(f"Title: {note[1]}")
    print(f"Body: {note[2]}")
    print(f"Created: {note[3]}")
    print("-" * 50)

conn.close()
```

**How to run it:**

1. Save the code to a file (e.g., `notes.py`)
2. Run it with: `python notes.py`

The script will:
- Create an in-memory SQLite database (disappears when the script ends)
- Create the notes table with your schema
- Insert 3 sample notes with timestamps
- Query and print all notes in a formatted way

If you want to persist the database to a file instead, replace `':memory:'` with a filename like `'notes.db'` in the `sqlite3.connect()` call.
