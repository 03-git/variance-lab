Here's a Python script that creates a SQLite database with a notes table, inserts sample data, and prints the results:

```python
import sqlite3
from datetime import datetime

# Create an in-memory database (or use 'notes.db' for a file)
conn = sqlite3.connect(':memory:')
cursor = conn.cursor()

# Create the notes table
cursor.execute('''
    CREATE TABLE notes (
        id INTEGER PRIMARY KEY,
        title TEXT,
        body TEXT,
        created_at TEXT
    )
''')

# Insert 3 sample notes
sample_notes = [
    ('Meeting Notes', 'Discussed project roadmap and Q2 goals', datetime.now().isoformat()),
    ('Shopping List', 'Milk, eggs, bread, cheese, vegetables', datetime.now().isoformat()),
    ('Ideas', 'New feature: dark mode toggle, improve search performance', datetime.now().isoformat())
]

cursor.executemany(
    'INSERT INTO notes (title, body, created_at) VALUES (?, ?, ?)',
    sample_notes
)
conn.commit()

# Query all notes
cursor.execute('SELECT id, title, body, created_at FROM notes')
notes = cursor.fetchall()

# Print the results
print("All Notes:")
print("-" * 80)
for note in notes:
    note_id, title, body, created_at = note
    print(f"ID: {note_id}")
    print(f"Title: {title}")
    print(f"Body: {body}")
    print(f"Created: {created_at}")
    print("-" * 80)

conn.close()
```

**To run this script:**

```bash
python3 script.py
```

If you want to save it to a file instead of an in-memory database, replace:
```python
conn = sqlite3.connect(':memory:')
```

with:
```python
conn = sqlite3.connect('notes.db')
```

The script will output all three notes with their IDs, titles, bodies, and creation timestamps.
