Here's a complete solution for an HTTP endpoint that returns the current time in ISO 8601 format as a JSON object, using **Python** and **Flask**. The endpoint is accessible at `GET /time`.

---

### ✅ Code

```python
from flask import Flask, jsonify
from datetime import datetime, timezone

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_current_time():
    # Get the current time in UTC and format it as ISO 8601
    current_time = datetime.now(timezone.utc).isoformat()
    return jsonify({"time": current_time})

if __name__ == '__main__':
    app.run(debug=True)
```
