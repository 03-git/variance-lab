I'll show you a Python script that does this:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print("\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        
        if len(response.text) > 500:
            print("\n... (truncated)")
    
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <url>")
        sys.exit(1)
    
    fetch_url(sys.argv[1])
```

**How to run it:**

```bash
python script.py https://example.com
```

The script:
- Uses `requests.get()` with `allow_redirects=True` to follow redirects automatically
- Prints the final status code (after any redirects)
- Displays all response headers
- Shows the first 500 characters of the response body
- Includes error handling for network issues
- Has a 10-second timeout to prevent hanging

Make sure you have the `requests` library installed: `pip install requests`
