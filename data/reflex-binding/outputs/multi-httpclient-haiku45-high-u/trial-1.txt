Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        body_preview = response.text[:500]
        print(f"\nFirst 500 characters of body:")
        print(body_preview)
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

Or with a URL that redirects:
```bash
python script.py https://bit.ly/some-short-url
```

**What it does:**
- Takes a URL from the command line
- Makes a GET request with `allow_redirects=True` to follow any redirects
- Prints the final status code, all response headers, and the first 500 characters of the response body
- Includes error handling for network issues with a 10-second timeout

**Requirements:**
You'll need the `requests` library. Install it with:
```bash
pip install requests
```
