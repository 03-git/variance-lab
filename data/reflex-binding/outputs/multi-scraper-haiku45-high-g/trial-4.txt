I'll show you a Python script that fetches a URL and extracts hyperlinks. Here's the code:

```python
import urllib.request
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self.current_text = []
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            href = dict(attrs).get('href', '')
            self.current_text = []
    
    def handle_data(self, data):
        self.current_text.append(data.strip())
    
    def handle_endtag(self, tag):
        if tag == 'a':
            text = ' '.join(self.current_text).strip()
            self.links.append((href, text) if 'href' in locals() else None)

# Fetch and parse
url = 'https://example.com'
try:
    with urllib.request.urlopen(url) as response:
        html = response.read().decode('utf-8')
    
    parser = LinkExtractor()
    parser.feed(html)
    
    for href, text in parser.links:
        if href:
            print(f"href: {href}")
            print(f"text: {text}")
            print()
except Exception as e:
    print(f"Error: {e}")
```

Actually, here's a cleaner version using `requests` and `BeautifulSoup`, which is simpler and more reliable:

```python
import requests
from bs4 import BeautifulSoup

url = 'https://example.com'

try:
    response = requests.get(url)
    response.raise_for_status()
    
    soup = BeautifulSoup(response.text, 'html.parser')
    
    for link in soup.find_all('a'):
        href = link.get('href', '')
        text = link.get_text(strip=True)
        print(f"href: {href}")
        print(f"text: {text}")
        print()
except Exception as e:
    print(f"Error: {e}")
```

**To run it:**

1. Install dependencies:
   ```bash
   pip install requests beautifulsoup4
   ```

2. Save the script to a file (e.g., `extract_links.py`) and run:
   ```bash
   python extract_links.py
   ```

Or run it directly in a Python REPL by pasting the code.

The BeautifulSoup version is preferred—it handles HTML parsing more robustly than the standard library parser.
