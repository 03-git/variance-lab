I'll show you a Python script that fetches a webpage and extracts hyperlinks:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def fetch_and_extract_links(url):
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} link(s):\n")
        
        # Extract and print href and link text
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            
            # Handle relative URLs
            if href and not href.startswith(('http://', 'https://', 'mailto:', '#')):
                href = urljoin(url, href)
            
            # Print with fallback for empty text
            display_text = text if text else "(no text)"
            print(f"URL: {href}")
            print(f"Text: {display_text}\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the page: {e}")

if __name__ == "__main__":
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. Install required packages:
```bash
pip install requests beautifulsoup4
```

2. Run the script:
```bash
python script.py
```

**What it does:**
- Fetches the URL with a 10-second timeout
- Parses the HTML using BeautifulSoup
- Finds all `<a>` tags
- Extracts `href` and link text
- Converts relative URLs to absolute URLs
- Handles cases where links have no text

You can modify the URL in the `if __name__ == "__main__"` section or pass it as an argument if you want to reuse the function.
