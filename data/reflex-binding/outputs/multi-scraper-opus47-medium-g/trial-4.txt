Per reflex.1 — kernel primitives first: the fetch is a TCP socket speaking HTTP, exposed in the stdlib as `urllib.request` (no `requests` needed); HTML parsing is `html.parser.HTMLParser` (no `bs4` needed). Stdlib-only:

```python
#!/usr/bin/env python3
import sys
from urllib.request import urlopen, Request
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self._href = None
        self._text = []

    def handle_starttag(self, tag, attrs):
        if tag == "a":
            self._href = dict(attrs).get("href")
            self._text = []

    def handle_data(self, data):
        if self._href is not None:
            self._text.append(data)

    def handle_endtag(self, tag):
        if tag == "a" and self._href is not None:
            self.links.append((self._href, "".join(self._text).strip()))
            self._href = None

def main(url="https://example.com"):
    req = Request(url, headers={"User-Agent": "link-extractor/1.0"})
    with urlopen(req, timeout=10) as resp:
        charset = resp.headers.get_content_charset() or "utf-8"
        html = resp.read().decode(charset, errors="replace")
    parser = LinkExtractor()
    parser.feed(html)
    for href, text in parser.links:
        print(f"{href}\t{text!r}")

if __name__ == "__main__":
    main(*sys.argv[1:])
```

Run it:

```bash
python3 links.py                 # defaults to https://example.com
python3 links.py https://news.ycombinator.com   # or pass any URL
```

Notes:
- `example.com` has exactly one link (`https://www.iana.org/domains/example`), so output is one line — point it at a richer page to see more.
- Relative hrefs are printed as-is; wrap with `urllib.parse.urljoin(url, href)` if you want them absolute.
- No `pip install` — everything here is in the standard library.
