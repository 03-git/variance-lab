Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <url>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    body = response.text[:500]
    print(body)
    if len(response.text) > 500:
        print("...")
        
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python script.py https://example.com
```

The script uses the `requests` library (which follows redirects by default with `allow_redirects=True`). If you don't have it installed:

```bash
pip install requests
```

The script will print:
- The final status code after following any redirects
- All response headers
- The first 500 characters of the response body (with "..." if there's more)
