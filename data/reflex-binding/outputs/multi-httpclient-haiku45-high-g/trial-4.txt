Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python script.py "https://example.com"
```

Or with a redirect chain:

```bash
python script.py "https://httpbin.org/redirect-to?url=https://example.com"
```

The script uses the `requests` library (which follows redirects by default), so you'll need to install it if you don't have it:

```bash
pip install requests
```

If you prefer a version using only the standard library (no external dependencies), here's one with `urllib`:

```python
import sys
import urllib.request
import urllib.error

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    with urllib.request.urlopen(url, timeout=10) as response:
        print(f"Status Code: {response.status}")
        
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        body = response.read().decode('utf-8', errors='ignore')
        print(f"\nFirst 500 characters of body:")
        print(body[:500])
        
except urllib.error.URLError as e:
    print(f"Error: {e}")
    sys.exit(1)
```

The `requests` version is simpler and more reliable. The `urllib` version has no dependencies but has slightly more verbose error handling.
