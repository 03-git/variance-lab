I'll show you a Python script that does exactly that:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    body_preview = response.text[:500]
    print(body_preview)
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python script.py "https://example.com"
```

The script:
- Takes a URL from the command line
- Uses `requests.get()` with `allow_redirects=True` (default) to follow redirects
- Prints the final status code
- Prints all response headers in a readable format
- Prints the first 500 characters of the response body
- Handles errors gracefully with a timeout of 10 seconds

If you don't have `requests` installed, run:
```bash
pip install requests
```
