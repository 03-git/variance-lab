I'll show you a Python script that fetches a webpage and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        
        # Extract and print href and link text
        for link in links:
            href = link.get('href', '#')
            text = link.get_text(strip=True)
            print(f"Text: {text}")
            print(f"URL:  {href}")
            print()
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the URL: {e}")

if __name__ == "__main__":
    url = "https://example.com"
    extract_links(url)
```

**How to run it:**

1. **Install dependencies** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags and extracts the `href` attribute and visible text
- Prints each link with its URL and text

You can modify the `url` variable in the `if __name__ == "__main__"` section to scrape different websites.
