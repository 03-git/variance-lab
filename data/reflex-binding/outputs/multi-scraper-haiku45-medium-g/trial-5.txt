I'll show you a Python script that fetches a webpage and extracts hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', 'N/A')
            text = link.get_text(strip=True)
            print(f"URL: {href}")
            print(f"Text: {text}\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the URL: {e}")
    except Exception as e:
        print(f"Error parsing HTML: {e}")

if __name__ == "__main__":
    url = "https://example.com"
    extract_links(url)
```

**How to run it:**

1. Install required libraries:
```bash
pip install requests beautifulsoup4
```

2. Save and run the script:
```bash
python script.py
```

**What it does:**
- Fetches the HTML from the URL using `requests`
- Parses it with `BeautifulSoup`
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints each URL and its associated text

You can modify the `url` variable at the bottom to scrape any webpage. The script includes error handling for network issues and parsing problems.
