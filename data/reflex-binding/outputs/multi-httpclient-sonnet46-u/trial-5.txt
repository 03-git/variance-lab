```python
import sys
import requests

def main():
    if len(sys.argv) != 2:
        print(f"Usage: {sys.argv[0]} <url>", file=sys.stderr)
        sys.exit(1)

    url = sys.argv[1]
    response = requests.get(url, allow_redirects=True)

    print(f"Status: {response.status_code}")
    print("\nHeaders:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")

    print("\nBody (first 500 chars):")
    print(response.text[:500])

if __name__ == "__main__":
    main()
```

**Dependencies:** requires the `requests` library.

**Run it:**

```bash
# Install dependency if needed
pip install requests

# Save the script, then run:
python fetch.py https://example.com

# Or pipe inline without saving:

# ... paste script here

```

`requests` follows redirects by default (`allow_redirects=True` is the default for GET, shown explicitly here for clarity). If you'd rather use only the standard library with no dependencies, replace the `requests` call with:

```python
import urllib.request
with urllib.request.urlopen(url) as response:
    status = response.status
    headers = dict(response.headers)
    body = response.read(500).decode("utf-8", errors="replace")
```
