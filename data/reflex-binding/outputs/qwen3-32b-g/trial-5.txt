To build a minimal HTTP endpoint that returns the current time in ISO 8601 format as JSON, here's a Python implementation using Flask:

```python
# time_api.py
from flask import Flask, jsonify
from datetime import datetime
import pytz

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_current_time():
    # Get current time in UTC (ISO 8601 standard)
    current_time = datetime.now(pytz.utc).isoformat()
    return jsonify({"time": current_time})

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```

**How to run:**
1. Install dependencies:
```bash
pip install flask python-dotenv pytz
```

2. Execute the script:
```bash
python time_api.py
```

3. Access the endpoint:
```bash
curl http://localhost:5000/time
# Output: {"time": "2024-03-25T14:30:45.123456+00:00"}
```

**Key components:**
- Uses `datetime.now(pytz.utc)` for proper UTC timestamp
- `jsonify()` ensures proper Content-Type header
- Runs on port 5000 (change as needed)
- Follows ISO 8601 standard with timezone info

For production, consider:
1. Using a WSGI server (gunicorn)
2. Setting `debug=False`
3. Configuring reverse proxy (nginx)
