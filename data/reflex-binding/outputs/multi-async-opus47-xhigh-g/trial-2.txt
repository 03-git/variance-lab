```python
import asyncio

async def handle_client(reader, writer):
    addr = writer.get_extra_info('peername')
    while True:
        data = await reader.readline()
        if not data:
            break
        writer.write(data)
        await writer.drain()
    writer.close()
    await writer.wait_closed()

async def main():
    server = await asyncio.start_server(handle_client, '127.0.0.1', 8888)
    async with server:
        await server.serve_forever()

asyncio.run(main())
```

Run:
```
python3 server.py
```

Test from another terminal:
```
nc 127.0.0.1 8888
```
Type a line, get it echoed. Multiple `nc` sessions run concurrently against the same process.
