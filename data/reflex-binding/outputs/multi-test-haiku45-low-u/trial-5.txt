I'll show you a Python test suite for that function using `pytest`, which is the most common testing framework.

**The function:**
```python
def celsius_to_fahrenheit(c):
    return c * 9/5 + 32
```

**The test suite:**
```python
import pytest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit:
    def test_freezing_point(self):
        assert celsius_to_fahrenheit(0) == 32
    
    def test_boiling_point(self):
        assert celsius_to_fahrenheit(100) == 212
    
    def test_negative_forty(self):
        assert celsius_to_fahrenheit(-40) == -40
    
    def test_non_numeric_input(self):
        with pytest.raises(TypeError):
            celsius_to_fahrenheit("0")
```

**How to run it:**

1. **Install pytest** (if you don't have it):
   ```bash
   pip install pytest
   ```

2. **Save the code** to a file like `test_temperature.py`

3. **Run the tests**:
   ```bash
   pytest test_temperature.py
   ```

   Or for verbose output:
   ```bash
   pytest test_temperature.py -v
   ```

The test suite covers:
