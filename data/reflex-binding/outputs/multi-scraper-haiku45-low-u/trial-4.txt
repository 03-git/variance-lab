I'll show you a Python script that fetches a webpage and extracts all hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} links on {url}\n")
        print(f"{'href':<50} {'Link Text':<50}")
        print("-" * 100)
        
        # Extract and print each link
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            print(f"{href:<50} {text:<50}")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install required packages:**
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**What the script does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints them in a formatted table
- Includes error handling for network issues

**To use it with different URLs:**
Just change the URL in the last line: `extract_links("https://your-url.com")`
