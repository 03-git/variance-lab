Here's a FastAPI implementation (Python) — clean validation with Pydantic, no boilerplate:

**`main.py`**
```python
from fastapi import FastAPI, Request
from fastapi.responses import JSONResponse
from pydantic import BaseModel, field_validator, EmailStr
from pydantic import ValidationError
import re

app = FastAPI()

class UserIn(BaseModel):
    name: str
    email: str
    age: int

    @field_validator("name")
    @classmethod
    def name_not_empty(cls, v):
        if not v.strip():
            raise ValueError("name must not be empty")
        return v

    @field_validator("email")
    @classmethod
    def email_contains_at(cls, v):
        if "@" not in v:
            raise ValueError("email must contain @")
        return v

    @field_validator("age")
    @classmethod
    def age_in_range(cls, v):
        if not (1 <= v <= 150):
            raise ValueError("age must be between 1 and 150")
        return v

@app.post("/users", status_code=201)
async def create_user(request: Request):
    try:
        body = await request.json()
    except Exception:
        return JSONResponse(status_code=400, content={"error": "invalid JSON"})

    try:
        user = UserIn(**body)
    except ValidationError as e:
        errors = [{"field": err["loc"][0], "message": err["msg"]} for err in e.errors()]
        return JSONResponse(status_code=400, content={"errors": errors})

    return JSONResponse(status_code=201, content=user.model_dump())
```

**How to run:**

```bash

pip install fastapi uvicorn

```

---

**Test it:**

```bash
# Valid request → 201
curl -s -X POST http://localhost:8000/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Alice", "email": "alice@example.com", "age": 30}' | jq

# → {"name": "Alice", "email": "alice@example.com", "age": 30}

# Missing name → 400
curl -s -X POST http://localhost:8000/users \
  -H "Content-Type: application/json" \
  -d '{"email": "alice@example.com", "age": 30}' | jq

# → {"errors": [{"field": "name", "message": "Field required"}]}

# Bad email → 400
curl -s -X POST http://localhost:8000/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Alice", "email": "notanemail", "age": 30}' | jq

# → {"errors": [{"field": "email", "message": "Value error, email must contain @"}]}

# Age out of range → 400
curl -s -X POST http://localhost:8000/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Alice", "email": "alice@example.com", "age": 200}' | jq

# → {"errors": [{"field": "age", "message": "Value error, age must be between 1 and 150"}]}
```

---

**Key design notes:**

- Pydantic handles type coercion — if `age` is `"abc"` it returns a clear type error automatically
- Multiple validation errors are returned together in the `errors` array, not just the first one
- `request.json()` is parsed manually so you get a clean 400 on malformed JSON instead of a 422/500
- FastAPI's interactive docs are available at `http://localhost:8000/docs` for free once running
