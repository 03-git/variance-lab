```python
import pytest
from temperature import celsius_to_fahrenheit


def test_freezing():
    assert celsius_to_fahrenheit(0) == 32

def test_boiling():
    assert celsius_to_fahrenheit(100) == 212

def test_negative_forty_crossover():
    assert celsius_to_fahrenheit(-40) == -40

def test_non_numeric_raises():
    with pytest.raises(TypeError):
        celsius_to_fahrenheit("hot")
```

Assumes the function lives in `temperature.py`:

```python
def celsius_to_fahrenheit(c):
    return c * 9/5 + 32
```

Run with:

```
pip install pytest
pytest -v test_temperature.py
```

Note: `-40` is the one temperature where Celsius and Fahrenheit coincide. The non-numeric test passes because `"hot" * 9` produces a string and `str / 5` raises `TypeError`.
