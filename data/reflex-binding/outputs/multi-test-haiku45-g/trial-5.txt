I'll show you a Python test suite using `unittest` for the celsius_to_fahrenheit function.

```python
import unittest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):
    def test_zero_celsius(self):
        self.assertEqual(celsius_to_fahrenheit(0), 32)
    
    def test_boiling_point(self):
        self.assertEqual(celsius_to_fahrenheit(100), 212)
    
    def test_negative_forty(self):
        self.assertEqual(celsius_to_fahrenheit(-40), -40)
    
    def test_non_numeric_input(self):
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("not a number")

if __name__ == '__main__':
    unittest.main()
```

**How to run it:**

Save this as a file (e.g., `test_convert.py`) and run:
```bash
python test_convert.py
```

Or run with verbose output:
```bash
python test_convert.py -v
```

The test suite covers:
- **0°C** → 32°F (freezing point)
- **100°C** → 212°F (boiling point)
- **-40°C** → -40°F (the point where both scales meet)
- **Non-numeric input** → expects a `TypeError` when passing a string
