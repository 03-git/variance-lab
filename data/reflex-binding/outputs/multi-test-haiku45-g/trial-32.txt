Here's a Python test suite for the `celsius_to_fahrenheit` function using `pytest`:

```python
import pytest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

# Tests
def test_freezing_point():
    assert celsius_to_fahrenheit(0) == 32

def test_boiling_point():
    assert celsius_to_fahrenheit(100) == 212

def test_forty_below():
    assert celsius_to_fahrenheit(-40) == -40

def test_non_numeric_input():
    with pytest.raises(TypeError):
        celsius_to_fahrenheit("not a number")
```

**To run it:**

1. Install pytest if you don't have it:
   ```bash
   pip install pytest
   ```

2. Save the code to a file (e.g., `test_converter.py`) and run:
   ```bash
   pytest test_converter.py
   ```

   Or run with verbose output:
   ```bash
   pytest test_converter.py -v
   ```

The tests cover:
- **0°C** → 32°F (freezing point of water)
- **100°C** → 212°F (boiling point of water)
- **-40°C** → -40°F (the point where both scales meet)
- **Non-numeric input** → raises `TypeError` (string input fails the multiplication)
