I'll write a Python test suite for the `celsius_to_fahrenheit` function using `pytest`:

```python
import pytest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit:
    def test_freezing_point(self):
        assert celsius_to_fahrenheit(0) == 32
    
    def test_boiling_point(self):
        assert celsius_to_fahrenheit(100) == 212
    
    def test_negative_forty(self):
        assert celsius_to_fahrenheit(-40) == -40
    
    def test_non_numeric_input(self):
        with pytest.raises(TypeError):
            celsius_to_fahrenheit("not a number")
```

**To run the tests:**

```bash
# Install pytest if you haven't already
pip install pytest

# Run the tests
pytest test_celsius.py -v

# Or if you want to run with more verbose output
pytest test_celsius.py -v --tb=short
```

The test suite covers:
- **0°C** → 32°F (freezing point of water)
- **100°C** → 212°F (boiling point of water)
- **-40°C** → -40°F (the one temperature that's the same in both scales)
- **Non-numeric input** → raises `TypeError` (the function will naturally raise this when trying to multiply a string)
