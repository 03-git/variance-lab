Here's a complete Python test suite using `unittest`:

```python
import unittest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):
    def test_freezing_point(self):
        self.assertEqual(celsius_to_fahrenheit(0), 32)
    
    def test_boiling_point(self):
        self.assertEqual(celsius_to_fahrenheit(100), 212)
    
    def test_forty_below(self):
        self.assertEqual(celsius_to_fahrenheit(-40), -40)
    
    def test_non_numeric_input(self):
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("abc")

if __name__ == '__main__':
    unittest.main()
```

**To run it:**

```bash
python test_celsius.py
```

Or for verbose output:
```bash
python test_celsius.py -v
```

The suite tests:
- **0°C** → 32°F (freezing point)
- **100°C** → 212°F (boiling point)
- **-40°C** → -40°F (the point where both scales meet)
- **Non-numeric input** → raises `TypeError` when you try to multiply a string by a number
