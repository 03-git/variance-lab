I'll show you a Python test suite for the `celsius_to_fahrenheit` function using the built-in `unittest` framework:

```python
import unittest

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):
    def test_zero_celsius(self):
        self.assertEqual(celsius_to_fahrenheit(0), 32)
    
    def test_boiling_point(self):
        self.assertEqual(celsius_to_fahrenheit(100), 212)
    
    def test_forty_below(self):
        self.assertEqual(celsius_to_fahrenheit(-40), -40)
    
    def test_non_numeric_input(self):
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("not a number")

if __name__ == '__main__':
    unittest.main()
```

**How to run it:**

```bash
python3 test_celsius.py
```

python3 test_celsius.py -v

The test suite covers:
- **0°C** → 32°F (freezing point)
- **100°C** → 212°F (boiling point)
- **-40°C** → -40°F (same in both scales)
- **Non-numeric input** → raises `TypeError` (the function will naturally raise this when trying `"not a number" * 9/5`)
