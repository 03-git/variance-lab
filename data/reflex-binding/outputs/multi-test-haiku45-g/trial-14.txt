I'll show you a Python test suite using `unittest` for the `celsius_to_fahrenheit` function.

```python
import unittest

def celsius_to_fahrenheit(c):
    """Convert Celsius to Fahrenheit."""
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):
    
    def test_freezing_point(self):
        """0°C should be 32°F."""
        self.assertEqual(celsius_to_fahrenheit(0), 32)
    
    def test_boiling_point(self):
        """100°C should be 212°F."""
        self.assertEqual(celsius_to_fahrenheit(100), 212)
    
    def test_same_in_both_scales(self):
        """-40°C should be -40°F."""
        self.assertEqual(celsius_to_fahrenheit(-40), -40)
    
    def test_non_numeric_input(self):
        """Non-numeric input should raise TypeError."""
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("30")

if __name__ == '__main__':
    unittest.main()
```

**To run it:**

```bash
python test_file.py
```

Or for verbose output:
```bash
python test_file.py -v
```

The test suite covers:
- **0°C → 32°F** (freezing point)
- **100°C → 212°F** (boiling point)
- **-40°C → -40°F** (where scales intersect)
- **Non-numeric input** (string raises `TypeError`)

All four tests will pass with the function as written.
