I'll show you a comprehensive Python test suite for the `celsius_to_fahrenheit` function using `unittest`, the standard testing framework.

```python
import unittest

def celsius_to_fahrenheit(c):
    """Convert Celsius to Fahrenheit."""
    return c * 9/5 + 32

class TestCelsiusToFahrenheit(unittest.TestCase):
    
    def test_freezing_point(self):
        """0°C should equal 32°F"""
        self.assertEqual(celsius_to_fahrenheit(0), 32)
    
    def test_boiling_point(self):
        """100°C should equal 212°F"""
        self.assertEqual(celsius_to_fahrenheit(100), 212)
    
    def test_same_in_both_scales(self):
        """-40°C should equal -40°F"""
        self.assertEqual(celsius_to_fahrenheit(-40), -40)
    
    def test_non_numeric_input(self):
        """Non-numeric input should raise TypeError"""
        with self.assertRaises(TypeError):
            celsius_to_fahrenheit("25")

if __name__ == '__main__':
    unittest.main()
```

**How to run it:**

```bash
# Run all tests with verbose output
python -m unittest test_module.TestCelsiusToFahrenheit -v

# Or if you save it as a script and run directly
python script.py -v

# Run a single test
```
