**Python (stdlib only):**

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
from datetime import datetime, timezone
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == '/time':
            body = json.dumps({"time": datetime.now(timezone.utc).isoformat()}).encode()
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.end_headers()
            self.wfile.write(body)
        else:
            self.send_response(404)
            self.end_headers()

    def log_message(self, *args):
        pass  # silence access log

HTTPServer(('', 8080), Handler).serve_forever()
```

**Run:**
```bash
python3 time_server.py
curl http://localhost:8080/time
```

**Output:**
```json
{"time": "2026-05-10T14:23:01.456789+00:00"}
```

---

If you prefer something with a framework already on the machine, here's the Flask equivalent:

```python
from flask import Flask, jsonify
from datetime import datetime, timezone

app = Flask(__name__)

@app.get('/time')
def get_time():
    return jsonify({"time": datetime.now(timezone.utc).isoformat()})

if __name__ == '__main__':
    app.run(port=8080)
```

```bash
pip install flask
python3 time_server.py
```

The stdlib version has no dependencies and is ready to run as-is.
