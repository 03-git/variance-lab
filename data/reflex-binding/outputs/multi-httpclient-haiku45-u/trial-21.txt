Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print(f"\nHeaders:")
    for header, value in response.headers.items():
        print(f"  {header}: {value}")
    
    print(f"\nBody (first 500 characters):")
    body = response.text[:500]
    print(body)
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python script.py https://example.com
```

The script:
- Takes a URL from the command line
- Makes a GET request with `allow_redirects=True` (follows redirects by default)
- Prints the final status code, all response headers, and the first 500 characters of the response body
- Handles request errors gracefully with a 10-second timeout

You'll need `requests` installed (`pip install requests`), or you can replace it with the built-in `urllib` if you prefer no dependencies.
