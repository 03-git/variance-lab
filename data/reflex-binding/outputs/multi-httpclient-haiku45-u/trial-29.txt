I'll show you a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print("\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        if len(response.text) > 500:
            print("...")
            
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
# Install requests if you don't have it
pip install requests

# Run the script with a URL
python script.py https://example.com

# Or with any other URL
python script.py https://www.google.com
```

The script uses the `requests` library which automatically follows redirects by default. It prints the final status code, all response headers, and the first 500 characters of the response body. If the body is longer than 500 characters, it adds "..." to indicate there's more.
