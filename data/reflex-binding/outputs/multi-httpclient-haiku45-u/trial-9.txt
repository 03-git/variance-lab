Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True)
    
    print(f"Status Code: {response.status_code}")
    print("\nHeaders:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nBody (first 500 characters):")
    print(response.text[:500])
except requests.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

1. Save the script (e.g., as `fetch_url.py`)
2. Make sure you have `requests` installed:
   ```bash
   pip install requests
   ```
3. Run it with a URL:
   ```bash
   python fetch_url.py https://example.com
   ```

The script will:
- Follow redirects automatically (`allow_redirects=True` is the default)
- Print the final status code
- Display all response headers
- Show the first 500 characters of the response body
- Handle errors gracefully if the request fails
