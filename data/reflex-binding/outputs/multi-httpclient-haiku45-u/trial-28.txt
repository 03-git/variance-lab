Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error: {e}", file=sys.stderr)
    sys.exit(1)
```

**How to run it:**

```bash
python script.py https://example.com
```

The script uses the `requests` library (which you may need to install with `pip install requests` if it's not already available). It:

- Takes a URL as the first command-line argument
- Makes a GET request with `allow_redirects=True` to follow redirects automatically
- Prints the final status code, all response headers, and the first 500 characters of the response body
- Handles errors gracefully and exits with a non-zero code on failure
