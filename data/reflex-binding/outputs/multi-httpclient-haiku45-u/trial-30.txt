I'll show you a Python script that does this. Here's the code:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        if len(response.text) > 500:
            print("...")
            
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

1. Save the code to a file (e.g., `fetch_url.py`)
2. Make sure you have `requests` installed:
   ```bash
   pip install requests
   ```
3. Run it with a URL:
   ```bash
   python fetch_url.py "https://example.com"
   ```

The script:
- Uses `requests.get()` with `allow_redirects=True` to follow redirects automatically
- Prints the final status code after all redirects
- Displays all response headers
- Shows the first 500 characters of the response body
- Includes error handling for network issues
- Has a 10-second timeout to prevent hanging
