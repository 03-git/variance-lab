Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <url>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True, timeout=10)
    
    print(f"Status Code: {response.status_code}\n")
    
    print("Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print(f"\nBody (first 500 characters):")
    print(response.text[:500])
    
except requests.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
python script.py "https://example.com"
```

**Key features:**
- `requests.get()` with `allow_redirects=True` (default) follows all redirects automatically
- `timeout=10` prevents hanging on unresponsive servers
- Prints status code, all headers, and the first 500 characters of the response body
- Handles errors gracefully with a try/except block

**Example output:**
```

  Content-Length: 1234
  Server: nginx
  ...

Body (first 500 characters):

<html>
<head>
    <title>Example Domain</title>
    ...
```
