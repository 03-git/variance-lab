Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    """Fetch a URL and display status code, headers, and body preview."""
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        if len(response.text) > 500:
            print("... (truncated)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>", file=sys.stderr)
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
python script.py https://httpbin.org/redirect/2
```

The script uses the `requests` library which handles redirects automatically via `allow_redirects=True`. If you don't have requests installed, install it with:

```bash
pip install requests
```
