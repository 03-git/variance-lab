Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for key, value in response.headers.items():
        print(f"  {key}: {value}")
    
    print("\nFirst 500 characters of body:")
    body_preview = response.text[:500]
    print(body_preview)
    
except requests.RequestException as e:
    print(f"Error: {e}")
    sys.exit(1)
```

**To run it:**

```bash
python script.py https://example.com
```

**Requirements:**
You'll need the `requests` library. Install it if you don't have it:
```bash
pip install requests
```

**Alternative using only the standard library** (no external dependencies):

```python
import sys
import urllib.request

if len(sys.argv) != 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    with urllib.request.urlopen(url) as response:
        print(f"Status Code: {response.status}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        body = response.read().decode('utf-8', errors='ignore')
        print("\nFirst 500 characters of body:")
        print(body[:500])
        
except urllib.error.URLError as e:
    print(f"Error: {e}")
    sys.exit(1)
```

The `requests` version is simpler and handles edge cases better. The standard library version works without external dependencies—both follow redirects automatically by default.
