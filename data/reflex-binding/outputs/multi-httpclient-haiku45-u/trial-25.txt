Here's a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        body_preview = response.text[:500]
        print(f"\nFirst 500 characters of body:")
        print(body_preview)
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <url>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

Replace `https://example.com` with any URL you want to fetch. The script will:
1. Follow redirects automatically (`allow_redirects=True`)
2. Print the final status code
3. Display all response headers
4. Show the first 500 characters of the response body

**Requirements:**
Make sure you have the `requests` library installed:
```bash
pip install requests
```
