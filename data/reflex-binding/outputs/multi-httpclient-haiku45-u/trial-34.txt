I'll show you a Python script that does exactly that:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}\n")
        
        print("Response Headers:")
        for key, value in response.headers.items():
            print(f"  {key}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body_preview = response.text[:500]
        print(body_preview)
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

**How it works:**
- `requests.get()` with `allow_redirects=True` automatically follows redirects
- Prints the final status code after all redirects
- Iterates through headers and displays each one
- Shows the first 500 characters of the response body
