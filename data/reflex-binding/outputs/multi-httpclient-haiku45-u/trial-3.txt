I'll create a Python script that does what you need:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print("\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print("\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        if len(response.text) > 500:
            print("... (truncated)")
            
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <url>", file=sys.stderr)
        sys.exit(1)
    
    fetch_url(sys.argv[1])
```

**How to run it:**

```bash
python script.py https://example.com
```

**What it does:**
- Takes a URL as the first command-line argument
- Makes a GET request with `allow_redirects=True` to follow redirects automatically
- Prints the status code, all response headers, and the first 500 characters of the response body
- Includes error handling for network issues with a 10-second timeout
