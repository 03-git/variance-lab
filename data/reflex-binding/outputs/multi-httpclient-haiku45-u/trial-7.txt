Here's a Python script that does what you need:

```python
import sys
import requests

if len(sys.argv) < 2:
    print("Usage: python script.py <URL>")
    sys.exit(1)

url = sys.argv[1]

try:
    response = requests.get(url, allow_redirects=True)
    
    print(f"Status Code: {response.status_code}")
    print("\nResponse Headers:")
    for header, value in response.headers.items():
        print(f"  {header}: {value}")
    
    print(f"\nFirst 500 characters of body:")
    print(response.text[:500])
    
except requests.exceptions.RequestException as e:
    print(f"Error making request: {e}")
    sys.exit(1)
```

**How to run it:**

```bash
# First, install requests if you don't have it
pip install requests

# Then run with a URL
python script.py https://example.com
python script.py https://httpbin.org/status/301
```

The script uses the `requests` library which automatically follows redirects by default (you can see the final status code after all redirects). It prints:
- The final status code
- All response headers
- The first 500 characters of the response body as text

If there's an error (network issue, invalid URL, etc.), it catches the exception and prints an error message.
