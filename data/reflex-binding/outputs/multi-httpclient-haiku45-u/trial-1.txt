I'll show you a Python script that does this. Here's the code:

```python
import sys
import requests

def fetch_url(url):
    try:
        response = requests.get(url, allow_redirects=True, timeout=10)
        
        print(f"Status Code: {response.status_code}")
        print(f"\nResponse Headers:")
        for header, value in response.headers.items():
            print(f"  {header}: {value}")
        
        print(f"\nFirst 500 characters of body:")
        body = response.text[:500]
        print(body)
        
    except requests.exceptions.RequestException as e:
        print(f"Error fetching URL: {e}")
        sys.exit(1)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python script.py <URL>")
        sys.exit(1)
    
    url = sys.argv[1]
    fetch_url(url)
```

**How to run it:**

```bash
python script.py https://example.com
```

Or if you want to see redirects in action:

```bash
python script.py https://httpbin.org/redirect-to?url=https://httpbin.org/get
```

The script uses the `requests` library (which handles redirects automatically with `allow_redirects=True`), prints the final status code after following any redirects, shows all response headers, and displays the first 500 characters of the response body. It also includes error handling for connection issues and requires a URL as a command-line argument.
