I will provide you with two versions: one using **Python (Flask)**, which is extremely readable, and one using **Node.js (Express)**, which is industry standard for web services.

### Option 1: Python (using Flask)

This is the fastest way to get an API up and running with minimal code.

#### 1. The Code (`app.py`)
```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    # datetime.now().isoformat() generates the ISO 8601 string
    current_time = datetime.now().isoformat()
    return jsonify({"time": current_time})

if __name__ == '__main__':
    # Run the server on port 5000
    app.run(debug=True, port=5000)
```

#### 2. How to run it
1.  **Install Flask:** Open your terminal and run:
    ```bash
    pip install flask
    ```
2.  **Run the application:**
    ```bash
    python app.py
    ```
3.  **Test it:** Open your browser or use `curl` in a new terminal:
    ```bash
    curl http://127.0.0.1:5000/time
    ```

---

### Option 2: Node.js (using Express)

This is ideal if you are working in a JavaScript-based ecosystem.

#### 1. The Code (`app.js`)
```javascript
const express = require('express');
const app = express();
const PORT = 3000;

app.get('/time', (req, res) => {
    // new Date().toISOString() generates the ISO 8601 string
    res.json({
        time: new Date().toISOString()
    });
});

app.listen(PORT, () => {
    console.log(`Server is running on http://localhost:${PORT}`);
});
```

#### 2. How to run it
1.  **Initialize and install Express:** Open your terminal and run:
    ```bash
    npm init -y
    npm install express
    ```
2.  **Run the application:**
    ```bash
    node app.js
    ```
3.  **Test it:** Open your browser or use `curl`:
    ```bash
    curl http://localhost:3000/time
    ```

---

### Expected Output
Regardless of which language you choose, when you hit the endpoint, you will receive a response like this:

```json
{
  "time": "2023-10-27T10:30:45.123456"
}
```

### Which one should you use?
*   **Use Python/Flask** if you are doing data science, AI, or want the simplest possible syntax.
*   **Use Node.js/Express** if you are building a high-concurrency web application or are already comfortable with JavaScript.
