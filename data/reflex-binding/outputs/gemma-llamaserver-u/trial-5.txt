### Option 1: Python (using Flask)
Flask is a lightweight and very popular web framework for Python.

#### 1. The Code (`app.py`)
```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_time():
    # Get current time in ISO 8601 format
    current_time = datetime.now().isoformat()
    return jsonify({"time": current_time})

if __name__ == '__main__':
    # Run the server on port 5000
    app.run(port=5000)
```

#### 2. How to run it
1.  **Install Python** (if you haven't already).
2.  **Install Flask** via your terminal:
    ```bash
    pip install flask
    ```
3.  **Run the application**:
    ```bash
    python app.py
    ```
4.  **Test it**: Open your browser or use `curl` in a new terminal:
    ```bash
    curl http://127.0.0.1:5000/time
    ```

### Option 2: Node.js (using Express)
Express is the industry standard for building web servers in JavaScript/Node.js.

#### 1. The Code (`index.js`)
```javascript
const express = require('express');
const app = express();
const PORT = 3000;

app.get('/time', (req, res) => {
    // Get current time in ISO 8601 format
    const currentTime = new Date().toISOString();
    res.json({ time: currentTime });
});

app.listen(PORT, () => {
    console.log(`Server is running at http://localhost:${PORT}`);
});
```

#### 2. How to run it
1.  **Install Node.js** (if you haven't already).
2.  **Initialize your project** and install Express in your terminal:
    ```bash
    npm init -y
    npm install express
    ```
3.  **Run the application**:
    ```bash
    node index.js
    ```
4.  **Test it**: Open your browser or use `curl` in a new terminal:
    ```bash
    curl http://localhost:3000/time
    ```
