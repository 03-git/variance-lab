**Endpoint Code**

We'll use Python with the Flask web framework to create the HTTP endpoint. Install Flask using pip:

```bash
pip install flask
```

Create a file named `app.py` with the following code:

```python
from flask import Flask, jsonify
from datetime import datetime

app = Flask(__name__)

@app.route('/time', methods=['GET'])
def get_current_time():
    """Return the current time as JSON"""
    current_time = datetime.now().isoformat()
    return jsonify({'time': current_time})

if __name__ == '__main__':
    app.run(debug=True)
```

**Explanation**

*   We import the necessary modules: Flask for creating the web application, `jsonify` for returning JSON data, and `datetime` for working with dates and times.
*   We create a Flask application instance, `app`.
*   We define a route for the `/time` endpoint, which responds to GET requests.
*   Inside the `get_current_time` function, we get the current time using `datetime.now()` and convert it to an ISO 8601 timestamp using `isoformat()`.
*   We return the current time as JSON using `jsonify`.
*   Finally, we run the application in debug mode using `app.run(debug=True)`.

**Running the Endpoint**

1.  Save the code in a file named `app.py`.
2.  Run the application using Python:

    ```bash
python app.py
```

    This will start the development server.
3.  Use a tool like `curl` to test the endpoint:

    ```bash
curl http://localhost:5000/time
```

    Replace `http://localhost:5000` with the actual URL where your Flask application is running.

**Tips and Variations**

*   To run the application on a specific port, use the `port` argument when calling `app.run()`: `app.run(port=5001, debug=True)`.
*   To deploy the application to a production environment, use a WSGI server like Gunicorn or uWSGI.
*   To expose the endpoint to the world, use a reverse proxy like NGINX or Apache.
