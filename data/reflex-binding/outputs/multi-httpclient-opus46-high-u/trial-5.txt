

```python
import sys
import urllib.request

url = sys.argv[1]
resp = urllib.request.urlopen(url)
print(f"Status: {resp.status}")
print("\nHeaders:")
for k, v in resp.getheaders():
    print(f"  {k}: {v}")
body = resp.read().decode("utf-8", errors="replace")
print(f"\nBody (first 500 chars):\n{body[:500]}")
```

**Usage:**

```bash
python script.py https://example.com
```

This uses only the standard library (`urllib.request`). `urlopen` follows redirects by default. If you want to use `requests` instead (better error handling, simpler API):

```python
import sys
import requests

resp = requests.get(sys.argv[1])
print(f"Status: {resp.status_code}")
print("\nHeaders:")
for k, v in resp.headers.items():
    print(f"  {k}: {v}")
print(f"\nBody (first 500 chars):\n{resp.text[:500]}")
```

This version requires `pip install requests` first.
