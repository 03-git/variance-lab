
```python
import sys
import urllib.request

def main():
    if len(sys.argv) != 2:
        print(f"Usage: {sys.argv[0]} <url>")
        sys.exit(1)

    url = sys.argv[1]
    req = urllib.request.Request(url, headers={"User-Agent": "Mozilla/5.0"})

    with urllib.request.urlopen(req) as response:
        print(f"Status: {response.status}")
        print("\nHeaders:")
        for header, value in response.getheaders():
            print(f"  {header}: {value}")
        body = response.read(500).decode("utf-8", errors="replace")
        print(f"\nBody (first 500 chars):\n{body}")

if __name__ == "__main__":
    main()
```

**How to run it:**

```bash
python script.py https://example.com
```

`urllib.request.urlopen` follows redirects by default (up to a reasonable limit). No third-party dependencies needed — this uses only the standard library.
