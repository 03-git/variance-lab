

```python
import urllib.request
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self._href = None
        self._text = []

    def handle_starttag(self, tag, attrs):
        if tag == "a":
            self._href = dict(attrs).get("href")
            self._text = []

    def handle_data(self, data):
        if self._href is not None:
            self._text.append(data)

    def handle_endtag(self, tag):
        if tag == "a" and self._href is not None:
            self.links.append((self._href, "".join(self._text).strip()))
            self._href = None

with urllib.request.urlopen("https://example.com") as resp:
    html = resp.read().decode()

parser = LinkExtractor()
parser.feed(html)

for href, text in parser.links:
    print(f"{href}  ->  {text}")
```

**Run it:**

```
python3 script.py
```

Uses only the standard library (`urllib.request` + `html.parser`). If you want something shorter and don't mind a dependency, `beautifulsoup4` would reduce the parsing to `soup.find_all("a")`.
