Here's a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com"

# Fetch the webpage
response = requests.get(url)
response.raise_for_status()  # Raise an error for bad status codes

# Parse the HTML
soup = BeautifulSoup(response.content, "html.parser")

# Extract all anchor tags
links = soup.find_all("a")

# Print each link's href and text
for link in links:
    href = link.get("href")
    text = link.get_text(strip=True)
    print(f"URL: {href}")
    print(f"Text: {text}")
    print()
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Save and run** the script:
   ```bash
   python script.py
   ```

**What it does:**
- `requests.get()` fetches the webpage
- `BeautifulSoup` parses the HTML
- `find_all("a")` finds all anchor tags
- For each link, it extracts the `href` attribute and the link text
- `strip=True` removes extra whitespace from the text

**Optional: Handle empty/missing values**

If you want to skip links with missing hrefs or text:

```python
for link in links:
    href = link.get("href")
    text = link.get_text(strip=True)
    
    if href:  # Only print if href exists
        print(f"URL: {href}")
        print(f"Text: {text if text else '(no text)'}")
        print()
```
