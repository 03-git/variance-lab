I'll show you a Python script that fetches a webpage and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks from the HTML."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise exception for bad status codes
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            print(f"URL: {href}")
            print(f"Text: {text}")
            print("-" * 50)
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the URL: {e}")

if __name__ == "__main__":
    # Fetch example.com
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies:**
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**What the script does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags
- Extracts the `href` attribute (the URL) and link text
- Prints each link's URL and text in a readable format
- Handles errors gracefully

You can modify the URL in the `fetch_and_extract_links()` call to fetch different websites.
