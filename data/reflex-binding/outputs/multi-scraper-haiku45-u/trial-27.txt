I'll show you a Python script that fetches a URL and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return
    
    soup = BeautifulSoup(response.content, 'html.parser')
    
    links = soup.find_all('a')
    
    if not links:
        print("No links found.")
        return
    
    print(f"Found {len(links)} link(s):\n")
    for i, link in enumerate(links, 1):
        href = link.get('href')
        text = link.get_text(strip=True)
        print(f"{i}. href: {href}")
        print(f"   text: {text}\n")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**To run it:**

1. Install dependencies:
```bash
pip install requests beautifulsoup4
```

2. Run the script:
```bash
python script.py
```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML and find all `<a>` tags
- Extracts the `href` attribute and link text from each anchor
- Prints them in a numbered list with both the URL and the display text

For `https://example.com`, you'll see example.com's domain information links printed out.
