Here's a Python script that fetches a URL and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return
    
    soup = BeautifulSoup(response.content, 'html.parser')
    links = soup.find_all('a')
    
    if not links:
        print("No links found")
        return
    
    print(f"Found {len(links)} link(s):\n")
    for link in links:
        href = link.get('href', 'No href')
        text = link.get_text(strip=True) or 'No text'
        print(f"Text: {text}")
        print(f"URL:  {href}\n")

if __name__ == '__main__':
    extract_links('https://example.com')
```

**How to run it:**

1. **Install dependencies:**
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**Output example:**
```

Text: More information...
URL:  https://www.iana.org/domains/example

Text: No text
URL:  https://www.iana.org/
```

**Key libraries:**
- `requests` — fetches the webpage
- `BeautifulSoup` — parses HTML and extracts elements

You can modify the `extract_links()` call to fetch different URLs, or add command-line arguments if you want to make it more flexible.
