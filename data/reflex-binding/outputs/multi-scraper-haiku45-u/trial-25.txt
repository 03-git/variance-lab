I'll show you a Python script that fetches a webpage and extracts all hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        print(f"{'URL':<50} | {'Link Text':<30}")
        print("-" * 82)
        
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            
            # Convert relative URLs to absolute
            if href:
                absolute_url = urljoin(url, href)
            else:
                absolute_url = "(no href)"
            
            print(f"{absolute_url:<50} | {text:<30}")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the URL: {e}")

if __name__ == "__main__":
    # Change this URL to any website you want to scrape
    url = "https://example.com"
    extract_links(url)
```

**How to run it:**

1. **Install required packages:**
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from the specified URL
- Parses it with BeautifulSoup
- Extracts all `<a>` tags and their `href` and text content
- Converts relative URLs to absolute URLs
- Displays results in a formatted table

**To modify it for different URLs:**
Change the `url` variable in the `if __name__ == "__main__":` section to any website you want to scrape.
