I'll show you a Python script that fetches a URL and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks from the HTML."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise an error for bad status codes
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} links on {url}\n")
        print(f"{'Link Text':<50} {'URL':<60}")
        print("-" * 110)
        
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            # Use empty string if text is missing
            text = text if text else "[no text]"
            print(f"{text:<50} {href:<60}")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    # Fetch and extract links from example.com
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from the URL using `requests`
- Parses the HTML with `BeautifulSoup`
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints results in a formatted table showing both the link text and URL

The script includes error handling for network issues and handles cases where links might not have text content.
