I'll show you a Python script that fetches a website and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Links found on {url}:\n")
        for link in links:
            href = link.get('href', '#')
            text = link.get_text(strip=True)
            print(f"Text: {text}")
            print(f"URL:  {href}\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the URL: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints each link's text and URL

If you want to modify it to fetch a different URL, just change `"https://example.com"` to your target URL.
