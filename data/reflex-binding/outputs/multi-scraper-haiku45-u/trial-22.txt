I'll show you a Python script that fetches a website and extracts hyperlinks:

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin

def extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the page
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Extract all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', '')
            text = link.get_text(strip=True)
            print(f"Text: {text}")
            print(f"URL:  {href}\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the page: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install required packages** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from `https://example.com`
- Parses it with BeautifulSoup
- Finds all `<a>` tags and prints their `href` attributes and link text
- Handles errors gracefully (network issues, timeout, etc.)

You can modify the URL by changing the argument to `extract_links()` on the last line, or make it accept command-line arguments if you'd like.
