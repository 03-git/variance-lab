Here's a Python script that fetches a website and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return
    
    soup = BeautifulSoup(response.content, 'html.parser')
    
    links = soup.find_all('a')
    
    if not links:
        print("No links found.")
        return
    
    print(f"Found {len(links)} link(s):\n")
    
    for link in links:
        href = link.get('href')
        text = link.get_text(strip=True)
        print(f"Text: {text}")
        print(f"URL:  {href}")
        print()

if __name__ == "__main__":
    extract_links("https://example.com")
```

**To run this:**

1. **Install dependencies** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags
- Extracts the `href` attribute (URL) and the link text
- Prints them in a readable format

**Example output:**
```
Found 2 link(s):

Text: Example Domain
URL:  https://www.iana.org/domains/example

Text: More information...
URL:  https://www.iana.org/domains/example
```

You can modify the URL by changing the string passed to `extract_links()` at the bottom of the script.
