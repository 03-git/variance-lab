I'll show you a Python script that fetches a webpage and extracts hyperlinks:

```python
import requests
from html.parser import HTMLParser
from urllib.parse import urljoin

class LinkExtractor(HTMLParser):
    def __init__(self, base_url):
        super().__init__()
        self.base_url = base_url
        self.links = []
        self.current_text = []
        self.in_anchor = False
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            self.in_anchor = True
            self.current_text = []
            for attr, value in attrs:
                if attr == 'href':
                    self.href = value
    
    def handle_data(self, data):
        if self.in_anchor:
            self.current_text.append(data.strip())
    
    def handle_endtag(self, tag):
        if tag == 'a' and self.in_anchor:
            self.in_anchor = False
            link_text = ' '.join(self.current_text)
            absolute_url = urljoin(self.base_url, self.href)
            self.links.append((absolute_url, link_text))

def extract_links(url):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        parser = LinkExtractor(url)
        parser.feed(response.text)
        
        return parser.links
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")
        return []

if __name__ == '__main__':
    url = 'https://example.com'
    links = extract_links(url)
    
    if links:
        print(f"Found {len(links)} links:\n")
        for href, text in links:
            print(f"URL: {href}")
            print(f"Text: {text or '(no text)'}")
            print()
    else:
        print("No links found or error occurred.")
```

**How to run it:**

1. **Install the required dependency:**
   ```bash
   pip install requests
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses `HTMLParser` to parse HTML and extract anchor tags
- Converts relative URLs to absolute URLs using `urljoin`
- Prints each link's href and link text
- Handles errors gracefully if the fetch fails

The script will print output like:
```
Found 5 links:

URL: https://example.com/page1
Text: Click here

URL: https://www.iana.org/domains/example
Text: More information...
```
