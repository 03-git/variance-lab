Here's a Python script that fetches a URL and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com"

try:
    response = requests.get(url, timeout=10)
    response.raise_for_status()
    
    soup = BeautifulSoup(response.content, "html.parser")
    
    links = soup.find_all("a")
    
    if not links:
        print("No links found on the page.")
    else:
        print(f"Found {len(links)} link(s):\n")
        for link in links:
            href = link.get("href")
            text = link.get_text(strip=True)
            print(f"URL: {href}")
            print(f"Text: {text}")
            print()
            
except requests.exceptions.RequestException as e:
    print(f"Error fetching the URL: {e}")
```

**How to run it:**

1. **Install dependencies** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from the URL using `requests`
- Parses the HTML with `BeautifulSoup`
- Finds all `<a>` tags
- Extracts the `href` attribute and link text for each link
- Prints them in a readable format
- Handles errors gracefully

You can modify the `url` variable to fetch any website you want. The script will work with any valid HTML page.
