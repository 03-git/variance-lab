Here's a Python script that fetches a URL and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    try:
        # Fetch the page
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} links:\n")
        for link in links:
            href = link.get('href', 'No href')
            text = link.get_text(strip=True) or 'No text'
            print(f"Text: {text}")
            print(f"URL:  {href}\n")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching the page: {e}")

if __name__ == "__main__":
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies:**
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script:**
   ```bash
   python script.py
   ```

**What it does:**
- Uses `requests` to fetch the HTML from the URL
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Prints each link with both the text and URL

**Output format:**
```
Text: Link text here
URL:  https://destination.com

Text: Another link
URL:  https://another.com
```

You can modify the URL in the `fetch_and_extract_links()` call to scrape any website.
