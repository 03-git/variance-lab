Here's a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print("No links found on the page.")
            return
        
        print(f"Found {len(links)} link(s):\n")
        for link in links:
            href = link.get('href')
            text = link.get_text(strip=True)
            print(f"  Text: {text}")
            print(f"  URL:  {href}")
            print()
    
    except requests.RequestException as e:
        print(f"Error fetching the page: {e}")

if __name__ == "__main__":
    extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if not already installed):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML and find all `<a>` tags
- Extracts both the `href` attribute and the visible link text
- Prints each link's text and URL
- Handles errors gracefully

**To fetch a different URL**, modify this line:
```python
extract_links("https://example.com")
```
