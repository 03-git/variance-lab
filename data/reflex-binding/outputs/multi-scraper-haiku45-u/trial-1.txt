I'll show you a Python script that fetches a URL and extracts hyperlinks. Here's the code:

```python
import requests
from bs4 import BeautifulSoup

def extract_links(url):
    """Fetch a URL and extract all hyperlinks from the HTML."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} links on {url}\n")
        print(f"{'Href':<50} | {'Link Text':<40}")
        print("-" * 92)
        
        for link in links:
            href = link.get('href', 'N/A')
            text = link.get_text(strip=True) or '[no text]'
            print(f"{href:<50} | {text:<40}")
    
    except requests.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    # Replace with the URL you want to scrape
    url = "https://example.com"
    extract_links(url)
```


**What the script does:**
- Uses `requests` to fetch the webpage
- Uses `BeautifulSoup` to parse the HTML
- Finds all `<a>` tags and extracts the `href` attribute and link text
- Displays results in a formatted table
- Handles errors gracefully (network issues, timeouts, etc.)
