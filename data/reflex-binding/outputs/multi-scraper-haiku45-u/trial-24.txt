I'll show you a Python script that fetches a URL and extracts hyperlinks from the HTML.

```python
import requests
from html.parser import HTMLParser

class LinkExtractor(HTMLParser):
    def __init__(self):
        super().__init__()
        self.links = []
        self.current_text = []
    
    def handle_starttag(self, tag, attrs):
        if tag == 'a':
            href = dict(attrs).get('href', '')
            self.current_text = []
    
    def handle_data(self, data):
        self.current_text.append(data.strip())
    
    def handle_endtag(self, tag):
        if tag == 'a':
            text = ' '.join(self.current_text).strip()
            self.links.append((href, text))

# Fetch and parse
try:
    response = requests.get('https://example.com', timeout=10)
    response.raise_for_status()
    
    parser = LinkExtractor()
    parser.feed(response.text)
    
    print("Links found:")
    print("-" * 80)
    for href, text in parser.links:
        print(f"href: {href}")
        print(f"text: {text}")
        print()
except requests.exceptions.RequestException as e:
    print(f"Error fetching URL: {e}")
```

**To run it:**

1. Install the required library:
   ```bash
   pip install requests
   ```

2. Run the script:
   ```bash
   python script.py
   ```

**Alternative using BeautifulSoup** (cleaner approach):

```python
import requests
from bs4 import BeautifulSoup

try:
    response = requests.get('https://example.com', timeout=10)
    response.raise_for_status()
    
    soup = BeautifulSoup(response.text, 'html.parser')
    
    print("Links found:")
    print("-" * 80)
    for link in soup.find_all('a'):
        href = link.get('href', '')
        text = link.get_text(strip=True)
        print(f"href: {href}")
        print(f"text: {text}")
        print()
except requests.exceptions.RequestException as e:
    print(f"Error fetching URL: {e}")
```

**To run the BeautifulSoup version:**

```bash
pip install requests beautifulsoup4
python script.py
```

The BeautifulSoup version is simpler and more robust. Both handle missing `href` attributes and extract link text gracefully.
