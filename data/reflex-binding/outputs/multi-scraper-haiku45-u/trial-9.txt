Here's a Python script that fetches a website and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks."""
    try:
        # Fetch the page
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} links on {url}\n")
        print("Link Text | Href")
        print("-" * 80)
        
        for link in links:
            text = link.get_text(strip=True)
            href = link.get('href', '')
            
            # Only print if either text or href exists
            if text or href:
                print(f"{text:<40} | {href}")
    
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Save and run** (e.g., as `fetch_links.py`):
   ```bash
   python fetch_links.py
   ```

3. **Or run directly in Python**:
   ```python
   import requests
   from bs4 import BeautifulSoup
   
   response = requests.get("https://example.com")
   soup = BeautifulSoup(response.content, 'html.parser')
   
   for link in soup.find_all('a'):
       text = link.get_text(strip=True)
       href = link.get('href', '')
       print(f"Text: {text} | Href: {href}")
   ```

The script handles errors gracefully and displays links in a formatted table with link text and href side-by-side. You can modify the URL to fetch any website you want.
