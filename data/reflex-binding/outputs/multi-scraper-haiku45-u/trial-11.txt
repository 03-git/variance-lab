Here's a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com"

try:
    response = requests.get(url, timeout=10)
    response.raise_for_status()
except requests.RequestException as e:
    print(f"Error fetching {url}: {e}")
    exit(1)

soup = BeautifulSoup(response.content, 'html.parser')

links = soup.find_all('a')

if not links:
    print("No links found on the page.")
else:
    print(f"Found {len(links)} link(s):\n")
    for link in links:
        href = link.get('href')
        text = link.get_text(strip=True)
        print(f"Text: {text}")
        print(f"URL:  {href}")
        print()
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script**:
   ```bash
   python script.py
   ```

**What it does:**
- Fetches the HTML from `https://example.com`
- Parses the HTML with BeautifulSoup
- Finds all `<a>` tags
- Prints the link text and href for each one
- Includes error handling for network issues

**Note:** example.com is a special reserved domain, so it has minimal content. Try replacing the `url` variable with any real website to see more links. For example: `"https://github.com"` or `"https://python.org"`
