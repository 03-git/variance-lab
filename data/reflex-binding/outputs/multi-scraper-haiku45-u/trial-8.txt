Here's a Python script that fetches a webpage and extracts all hyperlinks:

```python
import requests
from bs4 import BeautifulSoup

def fetch_and_extract_links(url):
    """Fetch a URL and extract all hyperlinks from the HTML."""
    try:
        # Fetch the webpage
        response = requests.get(url, timeout=10)
        response.raise_for_status()  # Raise exception for bad status codes
        
        # Parse the HTML
        soup = BeautifulSoup(response.content, 'html.parser')
        
        # Find all anchor tags
        links = soup.find_all('a')
        
        if not links:
            print(f"No links found on {url}")
            return
        
        print(f"Found {len(links)} links on {url}\n")
        print("=" * 80)
        
        # Extract and print each link
        for link in links:
            href = link.get('href', 'No href')
            text = link.get_text(strip=True) or '(no text)'
            print(f"Text: {text}")
            print(f"URL:  {href}")
            print("-" * 80)
            
    except requests.exceptions.RequestException as e:
        print(f"Error fetching {url}: {e}")

if __name__ == "__main__":
    fetch_and_extract_links("https://example.com")
```

**How to run it:**

1. **Install dependencies** (if you don't have them):
   ```bash
   pip install requests beautifulsoup4
   ```

2. **Run the script** (save as `fetch_links.py` and run):
   ```bash
   python fetch_links.py
   ```

Or **run it directly in a Python shell**:
```python
import requests
from bs4 import BeautifulSoup

response = requests.get("https://example.com", timeout=10)
soup = BeautifulSoup(response.content, 'html.parser')

for link in soup.find_all('a'):
    print(f"Text: {link.get_text(strip=True) or '(no text)'}")
    print(f"URL:  {link.get('href', 'No href')}")
    print()
```

The script uses:
- **`requests`** – to fetch the webpage
- **`BeautifulSoup`** – to parse the HTML and find all `<a>` tags
- It handles errors gracefully and extracts both the `href` attribute and the visible link text
