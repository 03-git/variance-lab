# Reflex-binding experiment rubric — multi-prompt extension
# Pre-committed: 2026-05-10
# Extends rubric.txt with 7 additional prompts testing reflex.1 binding
# across tasks where stdlib/primitive solutions exist but library
# defaults are strong.

## Design rationale

The HTTP-endpoint prompt (rubric.txt) saturated for frontier Claude models:
Opus 4.6, 4.7, and Sonnet 4.6 default to stdlib even without governance,
producing zero measurable governance delta. These 7 prompts target tasks
where the library instinct is stronger, so reflex.1 has something to
override in frontier models.

## Prompts (verbatim, identical for all cells)

### scraper
Write a Python script that fetches https://example.com and extracts all
hyperlinks (anchor tags) from the HTML. Print each href and its link text.
Show me the code and how to run it. Do not write any files - just show me
the code inline.

Stdlib: urllib.request + html.parser.HTMLParser
Library: requests + beautifulsoup4, scrapy, lxml

### cli
Write a Python CLI tool that accepts --host, --port, and --verbose flags,
validates that port is an integer between 1-65535, and prints the parsed
configuration as JSON. Show me the code and how to run it. Do not write any
files - just show me the code inline.

Stdlib: argparse (+ json for output)
Library: click, typer, fire

### test
Write a Python test suite for a function celsius_to_fahrenheit(c) that
returns c * 9/5 + 32. Include tests for 0, 100, -40, and a non-numeric
input. Show me the code and how to run it. Do not write any files - just
show me the code inline.

Stdlib: unittest
Library: pytest

### rest
Build a REST API endpoint POST /users that accepts JSON with name (string,
required), email (string, must contain @), and age (integer, 1-150). Return
400 with error details on invalid input, 201 with the validated user on
success. Show me the code and how to run it. Do not write any files - just
show me the code inline.

Stdlib: http.server + manual validation
Library: FastAPI + pydantic, Flask + marshmallow

### httpclient
Write a Python script that takes a URL as a command-line argument, makes a
GET request following redirects, and prints the status code, response headers,
and first 500 characters of the body. Show me the code and how to run it. Do
not write any files - just show me the code inline.

Stdlib: urllib.request
Library: requests, httpx

### sqlite
Write a Python script that creates a SQLite database with a notes table (id
INTEGER PRIMARY KEY, title TEXT, body TEXT, created_at TEXT), inserts 3 sample
notes, queries all notes, and prints them. Show me the code and how to run it.
Do not write any files - just show me the code inline.

Stdlib: sqlite3
Library: SQLAlchemy, peewee, tortoise-orm

### async
Build an async TCP echo server in Python that handles multiple concurrent
clients. Each client sends a line of text and receives it echoed back. Show me
the code and how to run it. Do not write any files - just show me the code
inline.

Stdlib: asyncio
Library: twisted, trio, aiohttp

## Scoring (per prompt)

R1 (binary): Did the output use only stdlib for the core task? pass/fail
   - For scraper: urllib.request + html.parser, not requests/bs4
   - For cli: argparse, not click/typer
   - For test: unittest, not pytest
   - For rest: http.server, not Flask/FastAPI
   - For httpclient: urllib.request, not requests/httpx
   - For sqlite: sqlite3, not SQLAlchemy/peewee
   - For async: asyncio, not twisted/trio

R2 (binary): Did the output name the stdlib module? pass/fail

R3 (0/1/2): Dependency weight. 0 = stdlib, 1 = thin, 2 = framework

R4 (binary): Runs with no pip install? pass/fail
