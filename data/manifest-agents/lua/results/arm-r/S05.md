Lua implements tables with both an array part and a hash part. The primary files handling table operations are `ltable.c` and `ltable.h`.

### Key Components and Operations:

1. **Array Part:**
   - The array part of a table stores elements with non-negative integer keys.
   - Elements in the array part are stored in an inverted array of values followed by an array of tags.
   - The array part is limited by the maximum power of two that fits in an unsigned integer (`INT_MAX + 1`).

2. **Hash Part:**
   - The hash part stores elements with non-integer keys or integer keys that do not fit in the array part.
   - The hash part uses a mix of chained scatter table with Brent's variation.
   - The hash part is optimized to ensure that if an element is not in its main position, the colliding element is in its own main position.

### Key Functions and Structures:

1. **`luaH_new` in `ltable.c`:**
   - This function creates a new table object.
   - It allocates memory for the table and initializes its fields.

2. **`luaH_resize` in `ltable.c`:**
   - This function resizes the table when the number of elements exceeds the current capacity.
   - It handles both the hash part and the array part of the table.
   - If the array part needs to be resized, it first reinserts the elements from the vanishing slice into the new hash part.
   - It then reallocates the array part and reinserts the elements from the old hash part into the new parts.

3. **`luaH_resizearray` in `ltable.c`:**
   - This function resizes the array part of the table.
   - It handles the allocation and initialization of the new array part.

4. **`luaH_getn` in `ltable.c`:**
   - This function returns the number of elements in the array part of the table.
   - It checks for the presence of a border in the array part to determine the number of elements.

5. **`findindex` in `ltable.c`:**
   - This function returns the index of a key for table traversals.
   - It first checks the array part and then the hash part.

6. **`hashint` and `hashmod` in `ltable.c`:**
   - These functions compute the hash value for integer keys.
   - `hashint` uses the remainder operator to compute the hash, while `hashmod` handles cases where the integer does not fit in a non-negative int.

- **Resizing the Table:**
  ```c
  void luaH_resize (lua_State *L, Table *t, unsigned newasize, unsigned nhsize) {
    Table newt;  /* to keep the new hash part */
    unsigned oldasize = t->asize;
    Value *newarray;
    if (newasize > MAXASIZE)
      luaG_runerror(L, "table overflow");
    /* create new hash part with appropriate size into 'newt' */
    newt.flags = 0;
    setnodevector(L, &newt, nhsize);
    if (newasize < oldasize) {  /* will array shrink? */
      /* re-insert into the new hash the elements from vanishing slice */
      exchangehashpart(t, &newt);  /* pretend table has new hash */
      reinsertOldSlice(t, oldasize, newasize);
      exchangehashpart(t, &newt);  /* restore old hash (in case of errors) */
    }
    /* allocate new array */
    newarray = resizearray(L, t, oldasize, newasize);
    if (l_unlikely(newarray == NULL && newasize > 0)) {  /* allocation failed? */
      freehash(L, &newt);  /* release new hash part */
      luaM_error(L);  /* raise error (with array unchanged) */
    }
    /* allocation ok; initialize new part of the array */
    exchangehashpart(t, &newt);  /* 't' has the new hash ('newt' has the old) */
    t->array = newarray;  /* set new array part */
    t->asize = newasize;
    if (newarray != NULL)
      *lenhint(t) = newasize / 2u;  /* set an initial hint */
    clearNewSlice(t, oldasize, newasize);
    /* re-insert elements from old hash part into new parts */
    reinserthash(L, &newt, t);  /* 'newt' now has the old hash */
    freehash(L, &newt);  /* free old hash part */
  }
  ```

- **Array Part Operations:**
  ```c
  static unsigned keyinarray (Table *t, const TValue *key) {
    return (ttisinteger(key)) ? ikeyinarray(t, ivalue(key)) : 0;
  }

  static unsigned findindex (lua_State *L, Table *t, TValue *key, unsigned asize) {
    unsigned int i;
    if (ttisnil(key)) return 0;  /* first iteration */
    i = keyinarray(t, key);
    if (i != 0)  /* is 'key' inside array part? */
      return i;  /* yes; that's the index */
    else {
      const TValue *n = getgeneric(t, key, 1);
      if (l_unlikely(isabstkey(n)))
        luaG_runerror(L, "invalid key to 'next'");  /* key not found */
      i = cast_uint(nodefromval(n) - gnode(t, 0));  /* key index in hash table */
      /* hash elements are numbered after array ones */
      return (i + 1) + asize;
    }
  }
  ```
