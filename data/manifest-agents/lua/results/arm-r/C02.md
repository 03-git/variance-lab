### String Representation

Strings in Lua are represented using the `TSHRSTR` (short shared string) or `TLNGSTR` (long string) tags in the `TValue` structure. The `TValue` structure is defined in `lobject.h` and includes a `tt` field that represents the type tag.

- **Short Shared String (`TSHRSTR`)**: This is used for short strings that are shared among multiple references. The actual string data is stored in a shared string table.
- **Long String (`TLNGSTR`)**: This is used for longer strings that are not shared. The string data is stored directly in the `TValue` structure.

The `TValue` structure is defined as follows:

  unsigned char s;      /* size of value */

- **Table (`TTABLE`)**: This tag is used for tables that are not userdata. The table data is stored in a `Table` structure.
