One commit between `d4e860d` and `HEAD` — `3b835e7`. Here's what changed:

**`runtime/hosuni/hosuni.c` — commit `3b835e7`**

1. **Glyph cache** — added a 512-slot open-addressing hash cache (`gcache`) for rendered bitmaps. `hosuni_draw_char` now checks the cache before calling `stbtt_GetCodepointBitmap`, and frees bitmaps immediately after caching (the removed `stbtt_FreeBitmap` at line 454 moved into the cache insert path).

2. **CJK font fallback** — new `hosuni_font_load_fallback()` loads a secondary `stbtt_fontinfo` into `f->fallback_info`. Both `hosuni_char_width` and `hosuni_draw_char` fall back to it when the primary font returns glyph index 0. Main searches NotoSansCJK in several distro paths on startup.

3. **xkbcommon input** — all manual `key_to_printable()` / inline key-table switch blocks removed across `handle_ask_key`, `handle_run_key`, `handle_edit_key`, `handle_focused_key`. Replaced with `platform_key_utf8(code, utf8, sizeof(utf8))`, which delegates to xkbcommon. `platform_xkb_update(ev.code, ev.value)` is now called unconditionally in the main event loop before the shift-key guard (previously it was conditional).

4. **Log schema fix** — `hosuni_log_qa` collapsed to a single row per Q/A pair (query + response + timing on one line), with tab-sanitization applied to the query string as well. The separate `Q\t` and `A\t` rows are gone.

5. **Build line** updated to add `-lm -lxkbcommon` and `-I/usr/include/libdrm`.
