Two hooks exist in `runtime/hooks/`:

**`bash.sh`** — sources `subtract.sh`, sets `PROMPT_COMMAND` to run `__subtract_capture` before each prompt, and installs `command_not_found_handle` to intercept unrecognized commands.

**`zsh.sh`** — same core setup, but uses zsh idioms: adds `__subtract_capture` to `precmd_functions`, installs `command_not_found_handler` (zsh spelling), sets `nonomatch` so unmatched globs like `do?` pass as literals instead of failing, and returns 0 unconditionally (required because zsh prints "command not found" on non-zero exit from the handler).

Both intercept the same two events: **every prompt** (to capture last command output) and **unrecognized commands** (to route through `__subtract_handle`).
