- **Subtract runtime**: runs `install.sh` as the target user
- **Display launch**: writes a `.bash_profile` that starts `cage → cog → multiplying.html` on tty1
- **User services**: enables lingering (`loginctl enable-linger`) so systemd user services start at boot, then enables `subtract-httpd` (`:8070`) and `subtract-bridge` (`:8889`) via `systemctl --user`
