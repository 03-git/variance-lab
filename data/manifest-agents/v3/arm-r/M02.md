## Verification Commands (recurring pattern across files)

The canonical verification form appears in `key-rotation.txt`:
```
ssh-keygen -Y verify -f authorized_signers -I hodori@subtract.ing \
  -n subtract.ing -s <file>.sig < <file>
```
This same primitive is referenced in `subtract.7.txt` (Reflex 2), `emile.txt` (signing model), and `governance.conf.universal` (authority.signed gate).
