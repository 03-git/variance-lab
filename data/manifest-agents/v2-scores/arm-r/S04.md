The lookdown system connects to `hosuni.c` through a series of functions that handle user input, pattern matching, and command execution. Here is the call path from user input to command execution:

1. **User Input Handling:**
   - The user input is processed in `hosuni.c` through the `lookdown_check` function, which is called when the input is short enough to be a lookdown pattern (less than 120 characters).
   - This function is part of the `lookdown_check` function, which is triggered by the `handle_request` function in `runtime/hosuni.sh`.

2. **Pattern Matching:**
   - `lookdown_check` iterates over the `lookdown_table` to find a matching pattern using `fnmatch`.
   - If a match is found, it proceeds to execute the associated command.

3. **Command Execution:**
   - If a match is found, the command is executed using `hosuni_exec` function.
   - `hosuni_exec` forks a child process to execute the command using `execlp`.

    for (int j = 0; j < lookdown_count; j++) {
        if (lookdown_table[j].flags & LOOKDOWN_EXEC) continue;
        if (fnmatch(lookdown_table[j].pattern, lower, 0) == 0) {
            if (match_idx) *match_idx = j;
            FILE *fp = popen(lookdown_table[j].command, "r");
            if (!fp) return -1;
            int len = fread(output, 1, output_max - 1, fp);
            output[len] = '\0';
            pclose(fp);
            while (len > 0 && output[len - 1] == '\n') output[--len] = '\0';
            return len;
        }
    }
    return -1;
}

static void hosuni_exec(const char *command) {
    platform_yield();

    sigset_t block, old;
    sigemptyset(&block);
    sigaddset(&block, SIGTERM);
    sigaddset(&block, SIGINT);
    sigprocmask(SIG_BLOCK, &block, &old);

    pid_t pid = fork();
    if (pid == 0) {
        sigprocmask(SIG_SETMASK, &old, NULL);
        platform_close_child_fds();
        execlp("sh", "sh", "-c", command, NULL);
        _exit(127);
    }
    if (pid > 0)
        waitpid(pid, NULL, 0);

    sigprocmask(SIG_SETMASK, &old, NULL);
    platform_reclaim();
    platform_drain_input();
}

In summary, the user input is processed in `lookdown_check`, which matches the input against predefined patterns in `lookdown_table`. If a match is found and the pattern is marked for execution, the command is executed using `hosuni_exec`, which forks a child process to run the command.
