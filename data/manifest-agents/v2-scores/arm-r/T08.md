The lookdown system is defined primarily in `runtime/hosuni/hosuni.c`. This file contains the implementation of the lookdown system, including the `lookdown_table` structure and related functions such as `lookdown_load`, `pinned_load`, `pinned_save`, and `lookdown_check`.

The organization of the lookdown system within `hosuni.c` is as follows:

1. **Structure Definition**: The `lookdown_table` structure is defined, which includes fields for patterns, commands, flags, and source information.
2. **Loading Functions**: Functions to load lookdown patterns from files, such as `lookdown_load` and `pinned_load`.
3. **Saving Functions**: Functions to save pinned lookdown patterns to files, such as `pinned_save` and `starred_save`.
4. **Pattern Matching and Execution**: The `lookdown_check` function checks input against the lookdown table and executes commands if a match is found.

Additionally, the `lookdown.universal.tsv` file contains the actual patterns and commands that the lookdown system uses. This file is referenced in the `lookdown_load` function to populate the `lookdown_table`.

In summary, the lookdown system is primarily defined in `runtime/hosuni/hosuni.c`, with the patterns and commands stored in `lookdown.universal.tsv`.
