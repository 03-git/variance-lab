`squared.c` is a C source file that implements an HTTP/2 proxy for local inference multiplexing. Specifically, it handles both cleartext HTTP/2 (h2c) and TLS-encrypted HTTP/2 (h2) connections. Here are the key details:

- **Purpose**: It acts as an HTTP/2 proxy to multiplex inference requests from a local client to a backend server, such as `llama-server`.
- **Modes**:
  - **h2c (cleartext HTTP/2)**: Operates in cleartext mode, limited to loopback connections.
  - **h2 over TLS (ALPN negotiated)**: Supports connections over TLS, allowing it to handle connections from browsers and other TLS-capable clients.
- **Build Instructions**: The file includes build instructions that require linking against `libnghttp2` and optionally `libmbedtls` for TLS support.
- **Usage**: It can be run with a specified port and backend host, optionally including a certificate and key for TLS support.

The file does not perform any computation itself but rather acts as a proxy to route inference requests efficiently.
