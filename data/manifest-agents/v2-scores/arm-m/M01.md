The `lookdown` pattern of glob-match-then-dispatch is architecturally similar in the following parts of the codebase:

1. **`./runtime/lookdown.m2mini.tsv`**:
   - This file contains a series of glob patterns and corresponding commands, which are matched against user input to determine the appropriate action. For example:
     ```tsv
     *smurfs*	mpv sftp://media@acer1660ti/mnt/media/kids/movies/Smurfs\ -\ The\ Lost\ Village\ \(2017\)/Smurfs.The.Lost.Village.2017.1080p.HULU.WEB-DL.DDP.5.1.H.264-PiRaTeS.mkv </dev/null >/dev/null 2>&1 & echo "playing smurfs"
     ```
   - The `lookdown_load` function in `./runtime/hosuni/hosuni.c` reads this file and processes the glob patterns and commands:
     ```c
     static void lookdown_load(const char *path) {
         FILE *fp = fopen(path, "r");
         if (!fp) return;
         int line_num = 0;
         char line[LOOKDOWN_PAT_MAX + LOOKDOWN_CMD_MAX + 8];
         while (fgets(line, sizeof(line), fp) && lookdown_count < MAX_LOOKDOWN) {
             line_num++;
             int len = strlen(line);
             if (len > 0 && line[len - 1] == '\n') line[--len] = '\0';
             if (len == 0 || line[0] == '#') continue;
             char *tab = strchr(line, '\t');
             if (!tab) continue;
             *tab = '\0';
             char *cmd = tab + 1;
             uint8_t flags = 0;
             if (cmd[0] == '[') {
                 char *close = strchr(cmd, ']');
                 char *tab2 = strchr(cmd, '\t');
                 if (tab2) {
                     if (close && close < tab2 &&
                         (strncmp(cmd, "[player]", 8) == 0 || strncmp(cmd, "[exec]", 6) == 0))
                         flags = LOOKDOWN_EXEC;
                     cmd = tab2 + 1;
                 } else continue;
             }
             if (!*cmd) continue;
             snprintf(lookdown_table[lookdown_count].pattern, LOOKDOWN_PAT_MAX, "%s", line);
             snprintf(lookdown_table[lookdown_count].command, LOOKDOWN_CMD_MAX, "%s", cmd);
             lookdown_table[lookdown_count].flags = flags;
             snprintf(lookdown_table[lookdown_count].source_file, 256, "%s", path);
             lookdown_table[lookdown_count].source_line = line_num;
             lookdown_count++;
         }
         fclose(fp);
     }
     ```

2. **`./lookdown.universal.tsv`**:
   - This file is another example of a lookdown pattern, containing a series of glob patterns and commands:
     ```tsv
     *square*feet*to*square*meter*	echo "Multiply square feet by 0.0929 to get square meters."
     *long*boil*egg*	echo "6 minutes for soft, 9 for medium, and 12 for hard-boiled."
     ```
