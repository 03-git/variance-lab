Two shell hooks exist in the `runtime/hooks/` directory:

1. `runtime/hooks/bash.sh`:
   - This hook sources the shared `subtract.sh` handler.
   - It sets up a `PROMPT_COMMAND` to capture the last command output before each prompt.
   - It defines a `command_not_found_handle` function to handle command-not-found events.

2. `runtime/hooks/zsh.sh`:
   - This hook also sources the shared `subtract.sh` handler.
   - It sets up a `precmd_functions` to capture the last command output before each prompt.
   - It defines a `command_not_found_handler` function to handle command-not-found events.

Both hooks are designed to integrate the `subtract` system into the respective shell environments (bash and zsh), allowing users to use natural language commands that are then translated into appropriate shell commands.
